## 예시 1: 전기 방석 — 소거법 + 제로마케팅

**타겟:** 안전에 민감한 어르신 선물 구매자 / 여름철 생리통·치질 환자
**선택 방식:** 소거법 + 제로마케팅
**핵심 인사이트:** 타겟 다각화로 계절 극복 (여름에도 하루 6개 이상 판매)

**콘티 구성:**
```
오프닝 (충격형):
"전기장판 중국산 전기 폭탄, 작년 사고 38,005건."
→ 뉴스 이미지 or 사고 통계 그래프
→ 즉각 해소: "그래서 이 제품은 다릅니다"

소거 단계:
저희 제품 안 사도 됩니다. 하지만 제대로 된 전기 방석 사기 위해선 알아야 합니다.
1. 리을자(ㄹ자) 열선의 문제: 고장 시 전체 기능 상실, 발열 느림
2. 저가 중국산의 문제: 사고 38,005건, 전자파·화재 위험
3. 유명 브랜드의 문제: 모델비 50%, 실제 기능 차이 없음

우리 제품 소개:
바코드형 열선 → 부분 고장 시 전체 작동 유지, 발열 빠르고 촘촘함
전자파 차단 인증, 국내 AS

타겟 확장:
"전기 방석, 겨울에만 쓰는 제품 아닙니다"
→ 생리통, 치질 환자의 온열 완화 욕구 자극 (여름에도 판매)

따뜻한 클로징:
"시장에 올바르지 못한 관행을 뒤로한 채, 저희는 오로지 제품에만 집중했습니다."
+ 100% 환불 보장
```

**원문 카피 (강사 멘트):**
> "전기장판 중국산 전기 폭탄, 작년 사고 38,005건."
> "저희 제품을 구매하지 않아도 좋습니다. 하지만 제대로 된 온풍기를 사기 위해선 알고 계셔야 됩니다."
> "한여름에도 하루 6개 이상 판매되는 이유가 있습니다."

## 예시 2: 키플 (거꾸리 성장기구) — 구매편익 + 죄책감 자극

**타겟:** 자녀 성장 걱정하는 부모 (구매자) / 성장기 자녀 (이용자)
**선택 방식:** 구매편익 (상징계 우선 + 죄책감 자극 + 매몰비용 활용)
**가격:** 40만원대 고가

**콘티 구성:**
```
오프닝 (죄책감 자극):
"우리 아이 성장기, 바라만 보고 계신가요?"
→ 부모의 죄책감 결핍 자극 (이용자≠구매자 전략)

상징계 (신뢰 구축, 1/2 점유):
"출시 10일, 13개 완판 (500만원 수익)"
"출시하자마자 환불 0건이 우리의 자신감입니다"
100% 환불 보장
→ 네이버 누적 후기 활용 ("오래 팔았던 제품의 후기"로 신뢰 이전)

기능계:
전문의 추천 + 성장 원리 설명
성장 전후 비교 GIF

매몰비용 활용 (클로징):
설치 후 반품 어려운 제품 특성 → 결심 유도
"오늘 설치하지 않으면 성장기를 놓칩니다"

사은품:
키플 벽걸이형 키재기 (15,000원 상당)
→ 욕구 직결: 성장을 직접 확인 → 사은품이 본제품 욕구 강화
```

**원문 카피:**
> "우리 아이 성장기, 바라만 보고 계신가요?"
> "출시하자마자 환불 0건이 우리의 자신감입니다."

**핵심 배울 점:**
- 이용자(자녀 성장) 기능보다 구매자(부모 죄책감) 욕구에 집중
- 사은품이 본제품 욕구와 정확히 연결됨 (키재기 → 성장 확인)
- 매몰비용(설치 후 반품 어려움)을 역으로 결심 유도에 활용

## 예시 3: 욕실 온풍기 — 소거법 단독

**타겟:** 욕실 온풍기 구매 고민 중인 일반 소비자
**선택 방식:** 소거법 (시장이 광고로 과열된 상황)

**콘티 구성:**
```
오프닝 (시장 문제 제기):
"지금 욕실 온풍기 시장은 광고로 과열되어 있는 거 알고 계시나요?"

교육적 진입:
"저희 제품을 구매하지 않아도 좋습니다."
"하지만 제대로 된 온풍기를 사기 위해선 알고 계셔야 됩니다."
"1, 2, 3, 4, 5." (소거 단계)

소거 1: 과장 광고 제품
소거 2: 저가 중국산
소거 3: (해당 시장 추가 소거 항목)

우리 제품 소개:
"시장에 올바르지 못한 관행을 뒤로한 채, 저희는 오로지 제품에만 집중했습니다."
→ 기능·안전 데이터

따뜻한 클로징:
"소비자를 기만하는 업체들에 맞서, 저희가 먼저 바꾸겠습니다."
+ 100% 환불 보장
```

**원문 카피:**
> "시장에 올바르지 못한 관행을 뒤로한 채, 저희는 오로지 제품에만 집중했습니다."
> "저희 제품을 구매하지 않아도 좋습니다. 하지만 제대로 된 온풍기를 사기 위해선 알고 계셔야 됩니다."

## 예시 4: 일반 제품 — 포지셔닝 방식 (곽철용 전략)

**선택 방식:** 포지셔닝 (신규 진입자, 유명하지 않을 때 가장 적합)

**콘티 구성:**
```
위 공격:
"아직도 연예인이 광고하는 가격 거품 가득한 브랜드만 고집하시나요?"
→ "모델비가 이 제품 값의 50%가 넘습니다"
→ "이름값을 여러분이 내고 계신 겁니다"

아래 공격:
"그저 싼 제품만 찾고 계신가요?"
→ "저가 중국산, AS 불가, 안전 증명서 없음"
→ "1년 안에 잔고장이 옵니다"

중간 진입:
"이래서 나왔습니다!"
→ "나도 살려고 새로 출시하여 후기수는 적지만 조작 없는 진짜 후기만 있는 M 브랜드입니다."
→ 핵심 가치 + 근거 자료 (데이터, 논문, 테스트 결과)

광고비 선언 (클로징):
"100원도 광고비 사용하지 않았습니다. 소비자의 만족으로만 광고하겠습니다."
+ 100% 환불 보장
```

**그래프 활용 예시 (뷔페 포지셔닝):**
```
X축: 거리 접근성 (가까울수록 위)
Y축: 가성비 (좋을수록 위)

시그니엘: 거리 접근성 중간, 가성비 낮음 → 오른쪽 하단
가평: 거리 접근성 낮음, 가성비 중간 → 왼쪽 중단
→ 신라 뷔페: 거리 접근성 좋음, 가성비 중간 → 이상적 위치
```

**주의:** 그래프 축 이름에 선호도 언어 사용 필수.
- ❌ "거리" (멀수록 위인지 아래인지 모름)
- ✅ "거리 접근성" (좋을수록 위)

## 예시 5: 바퀴벌레 퇴치기 (m트랩) — 스토리텔링

**타겟:** 반지하 거주자, 저소득층, 바퀴벌레 문제 심각한 가정
**선택 방식:** 스토리텔링 (화자 개입 + 타겟과 동일 처지 연출)

**콘티 구성:**
```
오프닝 (타겟 상황 연출):
반지하 사진
"살충제 써봤는데 머리만 아프고, 퇴치기는 효과도 없고..."

화자 개입:
"저도 같은 상황이었습니다. 화가 났어요."
"동공학에서 효과 있다는 걸 발견했는데, 왜 이걸 아무도 모르는 거죠?"

스토리 전개:
"m트랩은 광고를 많이 하지도 않고, 유명하지도 않습니다."
"쉽지도 않고, 그럴 자신도 없습니다."
"하지만 바퀴벌레 잘 잡는 것만은 자신 있습니다."

사회적 연결 (클로징):
"3개 팔릴 때마다 1개를 저소득층 가정에 후원합니다."
→ 타겟(반지하 거주자)이 "우리 편"으로 느낌
```

**성과:** 스토리 힘 덕분에 제품 설명 전에 신뢰도 구축 완료 → 이후 내용 자연스럽게 읽힘

**핵심 배울 점:**
- 타겟의 정확한 상황(반지하 사진)으로 시작 → 즉각적 공감 형성
- 약점 먼저 인정 ("유명하지 않습니다") → 역으로 신뢰 획득
- 사회적 기여가 타겟(저소득층)과 직결 → 사은품 이상의 감성 연결

## 예시 6: 서큘레이터 — 스토리텔링 (가격 포지셔닝)

**선택 방식:** 스토리텔링 오프닝 + 포지셔닝 결합

**오프닝:**
> "왜 서큘레이터는 선풍기와 비슷한 효과인데 10만원이 넘어야 할까요?"
> "우리는 생각했습니다. 서큘레이터 장점 더하면서 가격은 선풍기에 멈추면 어떨까?"

**구조:**
```
시장 빈틈 질문 ('왜'로 시작)
→ 기존 제품들의 문제 나열
→ 화자 고민·개발 과정
→ 새로운 가치 제시 ("장점은 그대로, 가격은 절반")
→ 클로징 (환불 보장)
```

**핵심:** '뭘 파는지'보다 '왜 파는지'에 집중하면 오프닝 프레임 자체가 바뀐다.

## 예시 7: 족저근막염 깔창 — 구매편익 (강력한 한 방)

**오프닝 (강력한 결핍 직격):**
> "발 통증 무조건 없어!"

**콘티 구성:**
```
상징계: 수만 명 사용 기록 + 전문의 추천 + 100% 환불
경험계 (경험재): GIF (발 통증 전후 비교)
         → "도대체 왜 계속 재구매하는 걸까요?"
         → 핵심 후기 3개 (소음, 내구성, 재구매 관련)
기능계: 발바닥 압력 분산 기술 설명 + 데이터
클로징: "지금 구매하시면 보행 교정 가이드 PDF 함께 드립니다"
```

## 실패 사례 — 하면 안 되는 것들

### 실패 1: 스킵존으로 시작

**실제 사례 (업소용 청소기 미션 피드백):**
```
❌ "업소용 청소기에 아쉬운 흡입력, 지치셨나요?
   오랫동안 사용할 수 있는 업소용 청소기에 관심이 많으신가요?"

→ 흡입력 때문에 검색해서 들어온 사람에게 의미 없는 질문
→ 접촉 지점이 가까우면 이런 서론은 즉시 스킵됨
→ 강사 평: "이게 뭔가 욕구에 와닿는 느낌이 없잖아요. 사실 크게 의미가 없다는 거죠."

✅ 해결: GIF로 흡입력 바로 시연 → 상징계(쇼핑몰 1위, 환불 보장) 진입
```

### 실패 2: 비판 후 해소 누락

**실제 사례 (노인용 보행기 미션 피드백):**
```
❌ "중국산 AS 안 됩니다, 안전 증명서 없는 제품을 경고했는데
   → 정작 우리 제품에 인증서 내용이 없음"

→ 강사 평: "우리가 명확하게 깠으면, 그 부분에 대한 해소가 들어와 있어야 한다고요.
   그래야 소비자들이 납득이 되죠."

✅ 해결: 비판 → "그래서 저희 제품은 KC인증·국내 AS 3년을 갖추고 있습니다" 즉시 제시
```

### 실패 3: 개연성 끊김

**실제 사례 (AB 슬라이더 미션 피드백):**
```
❌ 8번 카피: "여자분들이 선호하는 근육 1위가 복근이래요"
   10번 카피: "잠깐 고객님, 지금 구매하려는 AB 슬라이더가 안전한 제품입니까?"

→ 복근 관련 이야기 하다가 갑자기 안전 주의 → 개연성 끊김
→ 강사 평: "이 개연성이 되게 중요하고, 어디서 만나고 누가 타겟인지가 중요한 포인트예요."

✅ 해결: 복근 운동기구 → 다른 방법의 한계 → AB 슬라이더만의 안전한 방법으로 흐름 유지
```

### 실패 4: 화자 혼용

**실제 사례 (비문증 영양제 미션 피드백):**
```
❌ "저는 비문증을 갖고 있고..." (개인 스토리)
   ...
   "저희는 최선을 다하겠습니다" (갑자기 판매자 전환)

→ 강사 평: "화자가 개입될 때는 하나로 맞춰서 진행하시는 게 좋습니다. 바뀌면 안 돼."

✅ 해결: '저는'으로 시작했으면 끝까지 '저는' — 일관성 유지
```

### 실패 5: 당일 출고를 모든 제품에 적용

**실제 사례 (업소용 청소기 미션 피드백):**
```
❌ 업소용 청소기 상단에 "당일 출고 오후 2시까지 주문 완료 시"를 최상단 배치

→ 강사 평: "청소기가 그렇게 빨리 필요할까요? 타겟이 없어요."
→ 당일 출고가 핵심 가치가 되는 제품: 식품, 약, 신선식품 등 시간 민감 제품

✅ 해결: 제품 특성에 맞게만 사용. 시간 민감하지 않은 제품엔 불필요.
```

### 실패 6: 여러 욕구 동시 자극

```
❌ "복근 운동에 좋고, 허리 통증에도 좋고, 다이어트에도 좋습니다"
   → 중구난방, 한 축이 없음, 핵심이 흐려짐

✅ 복근 운동기구: 복근 욕구 하나에 집중
   강사 평: "핵심 가치를 잘 잡아야 한 축으로 내려갈 수 있어요. 그렇지 않으면 중구난방이 됩니다."
```
