# 포지셔닝·가치 전략 검수 체크리스트 (02_positioning)

> **로드 시점**: 포지셔닝 전략 결과물이 나온 직후 검수 단계에서만.
> 작업 시작 시 로드하지 않는다 (토큰 낭비).
> **성격**: 판단 여지 없는 확정 규칙만. 재발 가능성 있는 실수 패턴 기록.

---

## 확정 규칙

### R-01. CV 한 줄 명확성
CV(핵심가치)가 한 줄로 명확히 정리됐는지 확인.
"품질 좋고 가격도 합리적"류의 차별화 없는 서술은 위반.

- **위반 예**: "고품질 저가격 제품" — 모든 제품이 쓸 수 있는 말
- **통과 예**: "아픈 사람이 이미 병원·약을 써봤지만 포기 안 하는 시장 = 결핍 제로화 포지션"
- **근거**: CV가 뭉개지면 이후 상세페이지와 채널 전략이 방향 없이 흩어진다.

### R-02. 포지셔닝 독점성 확인
포지셔닝이 경쟁사와 명확히 구분되는 독점 포지션인지 확인.
"경쟁사보다 좋다"는 비교우위 서술은 포지셔닝이 아니다.

- **위반 예**: "경쟁사보다 품질이 좋음"
- **통과 예**: "경쟁사가 '기능'을 강조하는 동안, 우리는 '안전'을 독점"

### R-03. 가치더하기 구체성
가치더하기 포인트가 구체적 방법인지 확인.
"더 좋아보이게", "차별화" 수준의 막연한 서술은 위반.

- **위반 예**: "스토리를 더해서 차별화"
- **통과 예**: "귤 300원 → 3만원: '제주 할머니가 직접 수확한 귤'이라는 스토리 + 목상자 패키징"

### R-04. 단점→장점 전환 검토 여부
제품의 단점을 장점으로 전환할 가능성이 검토됐는지 확인.
단점이 없는 제품은 없다 — 검토 결과 "전환 불가"라도 명시해야 함.

### R-05. 자연적 현상 활용 검토 여부
자연적으로 일어나는 현상(계절, 신체 변화, 시간 흐름 등)을 제품 효과처럼 연결할 수 있는지 검토됐는지 확인.
검토 결과 "해당 없음"이라도 명시해야 함.

### R-06. 타겟별 CV 분리
01에서 넘어온 타겟이 2개 이상이면, 각 타겟별로 CV가 다르게 설정됐는지 확인.
타겟이 다른데 CV가 동일하면 위반 (타겟마다 핵심가치가 다를 수 있음).

### R-07. 01 결과와 일관성
01(결핍·타겟 분석)에서 넘어온 결핍 유형, 관여도, 구매편익과 포지셔닝 전략이 일관성 있는지 확인.
- 예: 01에서 "기능재"로 분류했는데 02에서 "상징재" 포지셔닝을 권하면 이유를 명시해야 함.

## 항목 추가 기준

다음 **모두** 만족할 때만 여기 추가:
- 동일/유사 실수가 **2회 이상** 발생
- **다른 제품·다른 세션에서 재발 가능** 한 표면 실수
- **판단 여지 없이 규칙으로 확정 가능** (맥락 판단이 필요하면 별도 메모)
