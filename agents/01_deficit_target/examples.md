## 예시 1. 캐리어 네임택 — 제로마케팅 + 불안 결핍

**타겟 발굴 사고 과정**
```
제품: 캐리어 네임택
1차 생각: "여행 많이 하는 사람" → 너무 넓다

결핍 파고들기:
- 여행 중 캐리어를 잃어버릴까봐 불안한 사람
- 1인 가구 여성 → 잃어버려도 찾아줄 사람 없음 = 불안 결핍
- 혼자 해외여행 → 영어로 신고해야 할 상황 = 불안 결핍

제로마케팅 프레임:
- 이미 가진 것: 캐리어, 여행 계획, 항공권(투자)
- 위협: 캐리어 분실
- 제품이 지키는 것: 이미 투자한 여행과 짐
```

**타겟 캐릭터**
```
직업: 직장인 또는 대학생
나이·성별: 20~30대 여성
생활 상황: 1인 가구, 연 1~2회 혼자 해외여행
핵심 결핍: 짐 분실 시 도움받을 사람 없음 + 영어 신고 두려움
간호도: 5~6 (여행 전에는 높아짐)
채널: 혼행(혼자 여행) 카페, 여행 커뮤니티, 인스타 여행 계정
```

**카피 방향**
- "혼자 해외여행 가는데 캐리어 잃어버리면?"
- "영어로 신고도 못하는데 짐이 사라졌다면?"
- "항공권까지 다 샀는데, 짐 분실로 망가진 여행은 아쉽잖아요"

**강사 핵심 멘트**
> "여행 갈 때 혼자라서 짐 맡길 사람도 없고, 잃어버리면 어쩌나 불안한 사람들. 1인 가구 여성. 걔네가 타겟이에요. 걔네는 이미 캐리어도 있고 여행 계획도 있어요. 근데 두려움이 있어. 그 두려움을 건드리는 거예요."

## 예시 2. 소음방지귀마개 — 동일 제품, 타겟별 간호도 비교

### 타겟 A: 조선소·목공소 종사자
```
핵심 결핍: 직업상 소음 노출
간호도: 3~4 (당연히 쓰는 거, 특별히 찾지 않음)
관여도: 낮음
경쟁 상황: 3M 헤드폰이 이미 선호 제품으로 자리잡음
채널: 조선소·목공소 카페
결론: 레드오션 + 경쟁 제품 강함 → 진입 어려움
```

### 타겟 B: 층간소음 피해자
```
핵심 결핍: 집에서 소음 피할 수 없음, 삶의 질 붕괴
간호도: 7~8 (새벽 6시에 글 올릴 정도로 절박)
관여도: 높음 (극단적 해결책까지 탐색)
채널: 층간소음 카페
실제 커뮤니티 언어:
  - "내 집에서 귀마개 해야 한다는 현타"
  - "발망치 사고 싶다"
  - "귀마개 추천해주세요" 글 다수
  - 새벽 6시 게시글, 조회수 높음
극단적 해결책 존재: 우퍼 구매, 층간소음 측정기
```

**등가교환 주의**: 간호도 7~8인데 귀마개 하나로 '해결'을 약속하면 이탈.
→ 셔플전략으로 접근 필요

**셔플전략 카피 방향**
```
"층간소음 완전 해결은 못 해도, 일단 오늘 밤 잠은 자야죠.
5가지 방법 중 귀마개부터 시작하는 이유"
```

**핵심 교훈**
간호도가 높다고 단순 해결책으로 들이밀면 안 된다.
고관여 타겟에게는 "여러 방법 중 하나"로 포지셔닝해야 한다.

## 예시 3. V라인밴드 — 욕구깡패 3단계 분석

**욕구 레이어 해체**
```
1차 욕구 (표면): "살 빼고 싶다" → 기능재 접근
2차 욕구: "예뻐지고 싶다" → 경험재 접근
3차 욕구 (욕구깡패): "남자친구/남편이 나를 다시 쳐다봐줬으면" → 관계 욕구
```

**카피 비교**
| 욕구 레이어 | 카피 | 반응 |
|-----------|------|------|
| 1차 (기능재) | "V라인 만들어주는 밴드" | 낮음 |
| 2차 | "예쁜 얼굴선을 만들어요" | 보통 |
| 3차 (욕구깡패) | "오늘부터 달라지면 걔가 다시 쳐다볼 거야" | 높음 |

**타겟 캐릭터 (욕구깡패 적용 버전)**
```
나이·성별: 30대 초반 여성
상황: 결혼 2~3년차, 남편이 요즘 무심해짐
핵심 욕구(3차): "남편이 예전처럼 나를 봐줬으면"
간호도: 6~7
채널: 맘카페, 부부 관련 카페
```

## 예시 4. GPS 위치추적기 — 제로마케팅 완전 적용

**제로마케팅 공식 적용**
| 타겟 | 이미 가진 것 | 잃을 위협 | 카피 방향 |
|------|------------|---------|---------|
| 아이 키우는 부모 | 아이 | 실종 | "잠깐 눈 돌린 사이, 아이가 없다면?" |
| 치매 부모 자녀 | 부모님 | 실종 | "어제도 혼자 나가셨는데 30분째 연락 없으면?" |
| 반려동물 집사 | 강아지·고양이 | 탈출 | "현관 열린 틈에 나간 뒤 못 찾으면?" |
| 차량 소유자 | 차 | 도난 | "블랙박스만 믿다가 차가 없어지면?" |

**강사 핵심 멘트**
> "제로마케팅은 새로운 걸 사게 하는 게 아니에요. 이미 가진 걸 잃지 않으려는 본능을 건드리는 거예요. 본전심리. 손실회피. 인간이 제일 무서워하는 게 잃는 거거든요."

## 예시 5. 온열찜질기 — 구매자 ≠ 이용자 전략

**기본 정보**
- 판매가: 39,800원 / 공급가: 10,000원 / 월 순익: 약 500만원
- 핵심 키워드: "여자친구 생리할 때"
- 경쟁사 판매가(27,900원)보다 비싼데도 CS 적음

**구매자·이용자 분리 분석**
```
이용자: 여자친구 → 생리통 완화 (기능 목적)
구매자: 20대 남자친구 → 여자친구에게 배려남으로 보이고 싶음

욕구깡패 (구매자):
1차: "여자친구 아픈 거 도와주고 싶다"
2차: "배려남으로 보이고 싶다"
3차: "이걸 계기로 더 깊은 관계가 되고 싶다"
```

**상세페이지 구성 원칙**
- 제품 스펙·효능 설명 비중 낮춤
- "이걸 선물받은 여자친구 반응" 중심으로 구성
- 카피: "생리통으로 힘들어하는 그녀에게 해줄 수 있는 가장 따뜻한 것"

**CS가 적은 이유**
구매자(남자친구)의 욕구(배려남 인정)가 선물 행위 자체로 충족됨
→ 여자친구가 제품에 불만이 있어도 CS로 이어지기 어려움
→ 경쟁사보다 비쌈에도 우위 가능

## 예시 6. 관절영양제 — 타겟 재설정 사례

### 실패 타겟
```
타겟: 어르신 (실제 구매자: 효도하려는 자녀)
채널: 효사모 카페
문제: 검색량 높지만 실구매로 이어지지 않음
원인:
  - 자녀의 효심이 온라인 구매까지 이어지기 어려움
  - 간호도 낮음 (어르신이 직접 절박하지 않음)
  - 구매자(자녀)와 이용자(어르신) 분리됐는데 욕구 연결 약함
```

### 성공 타겟 후보
| 타겟 | 핵심 결핍 | 간호도 | 등가교환 | 채널 |
|------|---------|------|--------|------|
| 피아노 전공자 | 손가락 관절 → 연주 못함 | 7 | 성립 | 피아노 커뮤니티 |
| 헬스·이종격투기인 | 관절 마모 → 운동 못함 | 6 | 성립 | 헬스 커뮤니티 |
| 수술 후 회복 환자 | 재활 중 영양 부족 | 8 | 성립 | 질병·재활 카페 |
| 테니스 동호인 | 테니스 엘보우 통증 | 6 | 성립 | 테니스 카페 |

**핵심 교훈**
제품이 안 팔릴 때 제품을 바꾸기 전에 타겟을 먼저 바꿔라.
결핍이 이미 있는 상태(수술 후)는 제로마케팅에 가까운 타겟이다.

## 예시 7. 배찜질기 — 등가교환 실패 경고

**타겟**: 불임 부부, 다낭성으로 배란 어려운 여성
**채널**: 불임 관련 카페
**간호도**: 7~8 (매우 높음, 절박함)

**왜 실패하는가**
```
타겟이 이미 시도한 것: 병원, 시술, 한의원, 보조제 수십 가지
배찜질기의 실제 해결력: 불임 해결 불가 (찜질 수준)
→ 타겟은 이미 전문가 수준의 지식 보유
→ "배찜질기가 도움된다"는 주장 → 즉시 신뢰 붕괴 → 이탈
```

**배찜질기의 적정 타겟**
- 간호도 3~4: 생리통, 소화 불량 등 일상적 불편을 가진 사람
- 간호도가 낮아 쉬운 해결책에도 반응하는 타겟

**핵심 공식**
`높은 간호도 타겟 + 낮은 해결력 제품 = 최악의 조합`

## 예시 8. 압박스타킹 — 타겟 캐릭터화 완성 예시

**완성된 페르소나**
```
직업: 서비스업 (승무원, 치과간호사, 카페 바리스타, 백화점 판매원, 비서)
나이·성별: 20~30대 초반 여성
생활 상황: 하루 8시간 이상 서서 근무, 퇴근 후 발·다리 부종
핵심 결핍: 부종으로 인한 통증 + 미관 불편 (다리 부어오름)
이미 시도한 해결책: 마사지, 족욕, 스트레칭
모이는 곳: 승무원 카페, 비서 카페 ('부종' 관련 글 다수)
간호도: 5~6
```

**카피 방향**
- "8시간 서있는 당신의 다리가 퇴근 때 얼마나 부어있는지 아세요?"
- "비서 선배들이 다 신는 이유가 있어요"

**채널 탐색 결과**
비서 카페에서 '부종'으로 검색 → 글 다수 → 실제 이야기로 광고 카피 작성

## 예시 9. 케겔운동기구 — 욕구 깊이로 타겟 분리

### 타겟 A: 출산 후 요실금 주부
```
간호도: 4~5 (불편하지만 일상생활 가능)
채널: 맘카페
접근: 무난한 건강 케어 포지셔닝
```

### 타겟 B: 섹스리스 아내 (1년 이상 잠자리 없음)
```
간호도: 7~8 (관계 위기 수준, 절박함)
채널: 부부 클리닉 카페, '제일 좋은 관계 연구소'
욕구깡패: "남편과의 관계를 회복하고 싶다" → 관계 욕구 (3차)
```

**타겟 B에게 케겔운동기구의 한계**
타겟 B의 간호도는 7~8인데, 케겔운동기구가 "부부 관계 회복"의 등가교환이 되는지 고민 필요.
→ 대안 제품 고려: 페로몬 향수 등 관계 개선에 더 직접적인 해결책

## 판단 체크리스트 (실전용)

제품이 주어지면 이 순서로 분석한다:

```
□ 이 제품이 해결하는 결핍은? (부족/불편/불안/욕망)
□ 기능재/경험재/상징재 분류
□ 제로마케팅 가능한가? (타겟이 이미 가진 것은?)
□ 욕구깡패: 3차 욕구까지 파고 들어라
□ 가장 간호도 높은 타겟 3개 도출
□ 각 타겟의 등가교환 성립 여부 확인
□ 구매자 ≠ 이용자인가?
□ 각 타겟이 모이는 채널 (커뮤니티명 구체적으로)
□ 타겟의 지적 수준·관여도 → 광고 톤 결정
□ 이상한 대안 힌트 1~2개
```
