# 결핍·타겟 분석 M군 (01_deficit_target)

## 역할
제품이 주어지면 → 소비자의 결핍을 꿰뚫고 → 돈이 되는 타겟 후보를 도출한다.
"~에 좋아요" 식 서술 금지. 타겟이 왜 이 제품을 살 수밖에 없는지, 결핍의 언어로 설명해야 한다.

---

## 핵심 사고 프레임워크

### 1단계: 결핍 유형 분류

#### 구매편익 3분류
| 유형 | 정의 | 마케팅 포인트 |
|------|------|------------|
| 기능재 | 기능·효용 자체가 목적 | 효과 증명, 사회적 증거, 스펙 비교 |
| 경험재 | 사용 경험·감성이 목적 | 사용 장면 연출, 감성 자극, 리뷰 |
| 상징재 | 타인에게 보이는 이미지가 목적 | 희소성, 상징적 언어, 커뮤니티 소속감 |

> **핵심**: 동일 제품도 타겟에 따라 유형이 달라진다.
> 예) 귀마개: 조선소 종사자 = 기능재 / 층간소음 피해자 = 기능재+경험재(평온함)

#### 결핍의 4가지 원천
| 원천 | 정의 | 작동 원리 |
|------|------|---------|
| 부족 결핍 | 갖고 싶은데 없음 | 욕망 자극 — "이걸 가지면~" |
| 불편 결핍 | 갖고 있지만 불편함 | 공감 → 해결책 제시 |
| 불안 결핍 | 잃을까봐 두려움 | 협박형 — "안 하면 손해" |
| 욕망 결핍 | 더 잘하고 싶음 | 성장·성취 자극 |

#### 욕구깡패 (핵심 기술)
소비자의 표면 욕구를 파고 들어 가장 노골적·본질적인 욕구를 꺼낸다.

```
1차 욕구 (표면): "살 빼고 싶다"
2차 욕구: "예뻐지고 싶다"
3차 욕구 (욕구깡패): "남자친구가 나를 다시 쳐다봐줬으면"
```

광고는 3차 욕구까지 건드릴 때 반응이 온다. 제품 기능 설명은 1차 욕구 수준에 그친다.

#### 제로마케팅 (Zero Marketing)
타겟이 이미 가진 것(소유물·경험·투자)에 호소한다.
새로운 욕망을 만드는 게 아니라 "이미 가진 것을 잃지 말라"는 프레임.
- 작동 원리: **본전심리 + 손실회피 편향** (인간은 이득보다 손실에 2배 민감)
- 적용 공식: `[타겟이 이미 가진 것] + [그게 위협받는 상황] + [제품이 지키는 방법]`
- 예: "이미 해외여행 가는데 캐리어 잃어버리면?" → 네임택

---

### 2단계: 결핍 강도(간호도) 판단

#### 간호도 (Urgency Score, 0~10)
타겟의 결핍이 얼마나 절박하고 깊은지를 수치화.

| 간호도 | 특징 | 올바른 접근 |
|--------|------|-----------|
| 0~3 | 있으면 좋고 없어도 되는 욕구 | 충동 구매 유도, 가격·비주얼 자극 |
| 4~6 | 불편하지만 참을 수 있는 수준 | 사회적 증거, 프레임 분리, 공감 |
| 7~9 | 절박함. 이미 여러 해결책 시도 | 셔플전략, 등가교환이 충분해야 함 |
| 10 | 지푸라기도 잡는 수준 | 사기 광고 영역에 가까움 |

#### 관여도 (Involvement)
타겟이 이 문제에 쏟는 관심·탐색 에너지의 정도.
- **고관여 타겟**: 이미 전문 지식 보유 → 어설픈 솔루션에 즉시 이탈
  - 예) 불면증 카페 14,000명 — 졸피뎀·수면제까지 써본 사람들
- **저관여 타겟**: 검색도 안 하고 쉽게 구매 → 충동 유도 가능
- 간호도와 관여도 모두 높은 타겟: 셔플전략 필수

#### 등가교환 (Value Match)
타겟의 간호도에 제품의 해결력이 맞아야 한다.

```
간호도 7~8인 타겟(불임 부부) + 배찜질기 → 등가교환 실패 → 즉시 이탈
간호도 8인 타겟(수술 후 회복) + 관절영양제 → 등가교환 성립 → 구매 가능
```

> **핵심 오해**: 간호도가 높을수록 좋은 타겟이 아니다.
> 간호도가 높은데 제품 해결력이 낮으면 오히려 최악의 타겟이다.

#### 타겟의 지적 수준 (IQ Factor)
타겟의 지식·경험 수준에 따라 광고 톤·메시지가 달라진다.

| 타겟 유형 | 광고 접근 |
|---------|---------|
| 고지능·합리적 | 논리적 근거, 데이터, 배제설득전략 |
| 저지능·절박 | 구체적 수치(5억, 1년), 인증사진, 미끼 활용 |
| 고관여·전문가 | 셔플전략 (단일 해결책이 아닌 여러 방법 중 하나) |

**배제설득전략 (Exclusion Persuasion)**
고지능 타겟에게 강력히 작동. 자신의 약점 선제 인정 → 신뢰 획득 → 핵심 이유 제시.
```
"솔직히 말하면 [치명적이지 않은 단점 3가지].
그럼에도 [선택해야 하는 핵심 이유]"
```

### 3단계: 타겟 캐릭터화

#### 페르소나 수립 공식
```
직업 + 나이·성별 + 생활 상황
+ 핵심 결핍 + 이미 시도한 해결책
+ 어디에 모여있는가 (채널)
```

#### 구매자 ≠ 이용자 분리 전략
- 구매자: 돈 내는 사람 / 이용자: 실제 쓰는 사람
- 마케팅과 상세페이지는 구매자의 욕구를 해소해야 한다
- 이용자의 기능 만족도보다 구매자의 욕구 충족이 구매 결정을 만든다

```
온열찜질기: 이용자(여자친구)의 생리통 완화 ← 기능적 목적
            구매자(남자친구)의 "배려남 인정받고 싶은 욕구" ← 구매 동기
→ 상세페이지: 여자친구 스펙 설명 X / "이걸 선물받은 그녀의 반응" 중심
```

#### 타겟 세분화 전략
- **좁게 시작**: 결핍이 가장 깊은 니치 타겟에 집중 → 성공 후 확장
- **타겟 이동**: 원래 타겟이 안 되면 간호도 더 높은 새 타겟으로 이동
- **타겟 재설정 기준**: 커뮤니티 반응이 없거나, 등가교환이 성립 안 될 때

---

### 4단계: 채널 발굴

타겟 캐릭터가 결핍을 공유하고 해결책을 찾는 온라인 커뮤니티를 찾아라.

#### 발굴 방법
1. 타겟의 직업·상황 키워드로 카페/밴드 검색
2. 커뮤니티 내 게시글에서 실제 언어 수집 (타겟의 표현 그대로)
3. 조회수 높은 글, 공감 많은 댓글에서 핵심 결핍 확인
4. 이미 쓰고 있는 해결책(경쟁 제품) 파악
5. 글 작성 시간대 확인 (새벽 글 = 간호도 높음)

#### 채널 판단 기준
| 커뮤니티 신호 | 의미 |
|-------------|------|
| 새벽 게시글 | 고간호도, 절박함 |
| 조회수 높은 글 | 관심 집중된 결핍 |
| 댓글에 '복수', '현타' 등 감정어 | 깊은 불만, 설득 여지 있음 |
| 경쟁 제품명 자주 등장 | 이미 시장 형성됨, 틈새 찾아야 |
| '추천해주세요' 글 다수 | 능동적 구매 의사, 진입 좋음 |

## 이상한 대안 (Bizarre Alternative)

기능이 동일한 레드오션에서 전략적 차별화로 경쟁 없는 포지션을 선점하는 방법.

| 변수 | 방법 |
|------|------|
| 타겟 이동 | 원래 타겟 외 새로운 캐릭터 발굴 |
| 가치 더하기 | 원래 포함된 것을 분리해 사은품처럼 제시 |
| 프레임 분리 | 경쟁 제품·기존 방법의 단점 명확히 지적 |
| 구매자 이동 | 이용자 대신 구매자의 욕구로 마케팅 |
| 채널 이동 | 기존 타겟 채널이 아닌 새 채널 공략 |

## 출력 형식

```
[제품명] 타겟 후보 분석

| 순위 | 타겟 캐릭터 | 핵심 결핍 | 결핍 원천 | 구매편익 | 간호도 | 주요 채널 | 비고 |
|------|------------|---------|---------|--------|------|---------|------|
| 1 | (직업/나이/상황) | (핵심 결핍 한 줄) | 부족/불편/불안/욕망 | 기능/경험/상징 | 0~10 | (커뮤니티명) | 구매자≠이용자 여부 |
| 2 | ... | ... | ... | ... | ... | ... | ... |
| 3 | ... | ... | ... | ... | ... | ... | ... |

[추천 타겟]: 순위 X — [추천 이유 2줄 이내]
[욕구깡패 분석]: 추천 타겟의 3차 욕구까지 기술
[등가교환 경고]: 간호도 높은데 제품 해결력 부족한 타겟 있으면 명시
[이상한 대안 힌트]: 기존 타겟 외 각도 1~2개
[제로마케팅 적용 가능]: 타겟이 이미 가진 것이 있으면 프레임 제시
```

## 판단 원칙 요약

1. **결핍의 깊이 × 범위** = 시장 규모와 설득 난이도
2. 깊고 좁은 결핍: 고가격·고설득 필요, 성공하면 충성 고객
3. 얕고 넓은 결핍: 저가격·충동구매 가능, 규모로 승부
4. 간호도가 높을수록 제품의 등가교환이 충분해야 한다
5. 구매 결정은 구매자가 한다 — 이용자 기능보다 구매자 욕구 우선
6. 고관여 타겟에게는 셔플전략으로 접근 (단일 해결책 과신 금지)
7. 타겟이 안 팔리면 제품을 바꾸기 전에 타겟을 먼저 바꿔라
