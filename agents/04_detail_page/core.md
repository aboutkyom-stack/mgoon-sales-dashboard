# 상세페이지 M군 (04_detail_page)

## 역할
타겟과 포지셔닝이 확정된 후 → 상세페이지 콘티(구성안)를 작성한다.
설득프로세스 5가지 기법(구매편익·소거법·이지선다·포지셔닝·스토리텔링) 중 제품·타겟·상황에 맞는 것을 선택해 스크롤별 섹션 순서를 결정한다.
"좋은 제품이에요" 나열 금지. 타겟의 결핍을 정확히 건드리고, 흐름이 막힘 없이 내려가야 한다.

---

## 핵심 사고 프레임워크

### 1단계: 설득 방식 선택

제품·타겟·시장 상황에 따라 5가지 방식 중 주 방식 1개를 선택하고, 보조 기법을 조합한다.

| 방식 | 언제 쓰나 | 핵심 구조 |
|------|---------|---------|
| 구매편익 | 판매자 신뢰가 핵심일 때, 경험재 제품 | 상징계(신뢰) → 기능/경험계 → 클로징 |
| 소거법 | 시장이 지저분하고 경쟁이 많을 때 | 나쁜 대안 제거 → 우리만 남음 → 따뜻한 클로징 |
| 이지선다 | 우리 제품의 특정 지점이 압도적으로 뛰어날 때 | 명확한 비교 → 우리가 훨씬 높음 |
| 포지셔닝 | 빠르게 상세페이지 만들어야 할 때, 신규 진입자 | 위·아래 공격 → "이래서 나왔습니다!" |
| 스토리텔링 | 브랜드 가치 구축, 장기 운영, 고가 제품 | 왜 파는지 → 시장 고발 → 화자 개입 → 개선 → 마무리 |

> **선택 기준**: 실력에 자신 있으면 이지선다+소거법. 시장이 더럽고 복잡하면 포지셔닝. 장기 브랜딩이면 스토리텔링. 판매자 신뢰가 먼저면 구매편익.

### 2단계: 섹션별 구성 원칙

#### 초반의 90% 법칙
상세페이지 초반 1/3~1/2이 전체 설득의 90%를 결정한다.
첫 카피에서 타겟의 핵심 결핍을 정확히 건드리지 못하면 스크롤이 멈춘다.

#### 스킵존 vs 리딩존

| 스킵존 (피해야 함) | 리딩존 (살려야 함) |
|-----------------|----------------|
| "이 제품에 관심 있으신가요?" | "중국산 전기장판, 작년 사고 38,005건" |
| "오랫동안 사용할 수 있는 청소기를 원하시나요?" | "발 통증 무조건 없어!" |
| 할인·이벤트·100% 보장 (뻔한 패턴) | 타겟 상황 직격탄 카피 |
| "이런 분들께 추천합니다" (접촉 지점 가까우면 불필요) | "모든 이들이 편안한 매트리스에서 잘 수 있을까?" |
| 제품 카테고리 설명 (이미 아는 정보) | 충격적 시장 현실 제시 |

> **핵심**: 접촉 지점이 이미 가까운 타겟(키워드 검색해서 들어온 사람)에게 서론 질문은 스킵존이 된다.

#### 섹션 구성 공식

```
[오프닝]      핵심 결핍 한 방 (리딩존 확보)
     ↓
[상징계]      신뢰 구축 (판매자 신뢰, 수상 경력, 완판 기록, 환불 보장)
     ↓
[제거/비교]   경쟁 제품·시장 문제 제기 → 즉각 해소 (개연성!)
     ↓
[기능계]      핵심 장점 2~3가지 (최대 5개) + 데이터·근거
     ↓
[경험계]      후기 3개 이내 (핵심 결핍 해소 후기만)
     ↓
[클로징]      구매 방해 요소 제거 (Q&A, 배송) + 행동 유도
```

### 3단계: 5가지 설득 방식 상세

#### 방식 1: 구매편익 (Purchase Benefit)

온라인 판매에서 판매자 신뢰가 제품 기능보다 먼저다.

**구조:**
```
상징계 (신뢰 구축, 상단 1/3~1/2 점유)
→ 한정 수량 / 완판 기록 / 키워드 1위 / 수상 경력
→ 100% 환불 보장 (위험 제거) — [판매자특성_선택 배열에 환불 보장 항목이 있는 경우에만 포함]
→ 할인 + 사은품 (욕구에 닿는 것, 가치 금액 명시)

기능재이면 → "반드시 ○○이어야 하는 이유 3가지" (데이터·스펙·비교)
경험재이면 → GIF로 메리트 직접 보여주기 + "도대체 왜 계속 사는 걸까요?" 후기
상징재이면 → 소속감·희소성 연출

클로징: 매몰비용 활용 / "지금 사면~"
```

**죄책감 자극 마케팅 (구매편익 변형):**
부모·자녀, 간병인·환자 등 책임감이 강한 구매자 타겟에 유효.
```
예) "우리 아이 성장기, 바라만 보고 계신가요?"
→ 결핍: 부모로서 아이에게 충분히 해주지 못한다는 죄책감
→ 해소: 이 제품 = 부모의 노력·사랑의 증거
```

#### 방식 2: 소거법 (Elimination Method)

시장의 나쁜 대안들을 논리적으로 제거해 우리만 남긴다.
**반드시 실력이 뒷받침될 때만 효과적.** 틈이 있으면 역공당한다.

**구조:**
```
교육적 진입:
"저희 제품을 구매하지 않아도 좋습니다.
 하지만 제대로 된 [카테고리]를 사기 위해선 알아야 할 것들이 있습니다."

나쁜 대안 1 제거 → 나쁜 대안 2 제거 → 나쁜 대안 3 제거
(각 제거 단계에서 틈이 없어야 함)

우리 제품 소개
→ 기능·데이터

따뜻한 클로징 (필수):
"시장에 올바르지 못한 관행을 뒤로한 채, 저희는 오로지 제품에만 집중했습니다."
```

**주의:** 소거 후 반드시 '따뜻함' 클로징. 안 하면 자신도 나빠 보인다.

---

#### 방식 3: 이지선다 (Either/Or)

특정 지점에서 우리가 압도적으로 뛰어날 때만 쓴다.

**구조:**
```
[경쟁 대안] vs [우리] 명확한 비교
→ 비교 대상이 우리보다 그 지점에서 현저히 낮아야 함
→ 중간 어중간한 대안이 없어야 함

예) "사업 경험 없는 컨설팅 vs 직접 사업해본 컨설팅"
    → 어느 쪽을 선택하겠습니까?
```

**강화 방법:** 비교 자체가 불가능할 정도의 극단적 대안을 제시하면 더 강력.

---

#### 방식 4: 포지셔닝 (Positioning)

가장 빠르게 만들 수 있는 상세페이지. 곽철용 전략.
`"A 안 되고, B 안 되고, 그 사이 C가 내 결론이다"`

**구조:**
```
위 공격: "아직도 연예인이 광고하는 가격 거품 가득한 브랜드만 고집하시나요?"
          → 비싼 브랜드의 단점 (모델비가 제품값 50%, 허상)

아래 공격: "그저 싼 제품만 찾고 계신가요?"
            → 저가 제품의 단점 (중국산, 안전 미검증, 잔고장)

중간 진입: "이래서 나왔습니다!"
            → 핵심 가치 + 근거 자료 (데이터, 논문, 테스트 결과)
            → 100% 환불 보장 — [판매자특성_선택 배열에 환불 보장 항목이 있는 경우에만 포함]
```

**그래프 포지셔닝:**
- X축·Y축은 반드시 선호도 언어로 설정
  - ❌ "거리" → ✅ "거리 접근성" (가까울수록 위)
  - ❌ "가격" → ✅ "가성비" (좋을수록 위)
- 경쟁사들을 양 끝에 배치 → 우리를 중간 이상적 위치에

**소거법과의 차이:**
- 소거법: 남기는 것이 중요 (논리적 제거 후 우리만)
- 포지셔닝: 그래프의 위치가 중요 (A, B 사이 우리가 들어갈 자리)

---

#### 방식 5: 스토리텔링 (Storytelling)

가장 강력하지만 가장 어렵다. 판매자를 "우리 편"으로 만든다.
**시장을 관통하지 못하면 어설픈 감정 호소로 전락한다.**

**구조:**
```
1. 시장 빈틈 '왜'로 시작:
   "왜 [카테고리]는 꼭 [현재 문제]여야만 했을까요?"
   "모든 이들이 [결핍 해소]를 할 수 있을까?"

2. 현 시장 상태 고발 (스토리텔링으로 나열):
   "A가 이렇고, B가 이렇고, 이게 현실입니다"

3. '나'라는 화자 개입:
   "저는 [자신의 경험/처지]를 [기간]간 겪으며..."
   "이 사람들은 한 번도 [이용자 처지]를 겪어보지 않은 것 같았습니다"

4. 문제 해결·개선의 여정:
   "그래서 저는 생각했습니다. [가치 제시]"

5. 새로운 문화 선언:
   "이 문화를 바꿔 나가려고 합니다"

6. 사회적 연결 (선택):
   "3개 팔릴 때마다 1개를 [수혜자]께 기부합니다"

7. 제품으로 자연스러운 전환
```

**화자 일관성 (절대 규칙):**
- "저는"으로 시작했으면 끝까지 "저는"
- "저희는"으로 시작했으면 끝까지 "저희는"
- 혼용 시 신뢰 즉시 붕괴

### 4단계: 콘티 작성법

상세페이지 = 콘티 + 디자인. 콘티가 먼저다.
허접해도 좋다. 흐름(논리 구조)이 맞으면 된다.

```
[콘티 형식]
섹션번호 | 섹션명 | 사진/이미지 구성 | 카피 방향 | 사용 기법

예시:
1 | 오프닝  | 전기장판 사고 뉴스 스크린샷  | "중국산 전기장판, 작년 사고 38,005건" | 충격→즉각 해소 연결
2 | 상징계  | 완판 뱃지 GIF              | "출시 3일, 완판 기록 / 환불 0건"      | 신뢰 구축
3 | 비교    | 바코드형 vs 리을자 열선 비교 | "바코드형 열선만이 부분 고장을 막습니다" | 소거법
4 | 기능계  | 열 분산 테스트 사진         | "38,005건 사고 이유, 우리가 없애는 법"  | 데이터 제시
5 | 경험계  | 핵심 후기 3개 캡처          | "도대체 왜 계속 사는 걸까요?"          | 경험재 후기
6 | 클로징  | 사은품 사진 + 구매 버튼      | "지금 주문하시면 핫팩 2개 함께"        | 행동 유도
```

**디자인 원칙:**
- 제목 폰트 ≠ 내용 폰트 (명확히 구별, 단 너무 두꺼우면 가독성 저해)
- 한 화면 3색 이하 (흰 배경 기본)
- 강조: 빨간 배경 + 흰 글씨, 또는 형광펜 반만 적용
- 충분한 여백 → 밀도 조절 → 가독성↑
- 비교 이미지: 한쪽 크기 줄이거나 음영 넣어 우월함 시각화
- 카피는 구어체로 ("뱉는 것"처럼). 의미 단위로 엔터 사용.

### 5단계: 사은품 전략

**황금 기준:** 별보다 배꼽이 큰 사은품 — 따로 사긴 애매하지만 받으면 가치 있는 것.

```
✅ 좋은 사은품 조건:
- 본제품 욕구와 연결됨 (온열찜질기 → 핫팩, 성장기구 → 키재기)
- 개별 구매하기엔 애매한 가격대 (5,000~20,000원)
- 사은품 자체 가치(금액) 반드시 명시 ("15,000원 상당")

❌ 나쁜 사은품:
- 본제품과 무관한 것
- 가치 금액 미표시
- 이미 흔한 사은품 (쇼핑백, 기프트카드)
```

## 판단 원칙 요약

1. **초반이 90%다** — 오프닝 카피가 결핍을 건드리지 못하면 나머지는 없다
2. 결핍을 제기했으면 반드시 즉각 해소하라 — 해소 없으면 이탈
3. 비판한 것에 대한 해결책이 반드시 나와야 한다 (개연성 법칙)
4. 한 축에 집중하라 — 여러 욕구 동시 자극 = 중구난방
5. 포지셔닝이 가장 쉽다. 스토리텔링은 가장 강력하지만 가장 어렵다
6. 디자인보다 카피+콘티(논리 구조)가 먼저다
7. 후기는 3개 이내, 핵심 결핍 해소 후기만
8. 화자 일관성: "저" 또는 "저희" 중 하나로만
