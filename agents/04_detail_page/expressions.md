## 1. 오프닝 패턴 (첫 카피 = 전쟁터)

오프닝이 실패하면 나머지는 없다. 타겟의 핵심 결핍을 첫 줄에서 건드려야 한다.

### A. 충격형 (소거법·제로마케팅 오프닝)
```
"[제품 카테고리], 작년 사고 [숫자]건 — 알고 계셨나요?"
"[경쟁 제품], 사실 [충격적 단점]인 거 아셨어요?"
"[시장], 지금 [문제 상황]으로 과열되어 있는 거 알고 계시나요?"

사례:
"중국산 전기장판, 작년 사고 38,005건."
"욕실 온풍기 시장, 지금 광고로 과열되어 있는 거 알고 계시나요?"
"N사 후기 3일 만에 천 개, 진짜일 것 같으세요?"
```

### B. 결핍 직격형 (구매편익 오프닝)
```
"[문제] 무조건 [해결]!"
"[타겟 상황] — 이제 끝냅니다"
"[죄책감 자극 질문]"

사례:
"발 통증 무조건 없어!"
"우리 아이 성장기, 바라만 보고 계신가요?"
"층간소음, 그냥 참고 계신가요?"
"[타겟 불편 상황], 지치셨나요?" → ❌ 접촉 지점 가까운 타겟엔 사용 금지
```

### C. 왜형 (스토리텔링 오프닝)
```
"왜 [제품 카테고리]는 꼭 [현재 문제]여야만 했을까요?"
"모든 이들이 [결핍 해소]를 할 수 있을까요?"

사례:
"왜 서큘레이터는 선풍기와 비슷한 효과인데 10만원이 넘어야 할까요?"
"모든 이들이 편안한 매트리스에서 잘 수 있을까?"
"왜 좋은 [카테고리]를 갖고 싶으면 무조건 비싸야만 할까요?"
```

### D. 포지셔닝 오프닝 (곽철용 화법)
```
"아직도 [비싼 브랜드] 고집하시나요?" + "그저 [싼 제품]만 찾고 계신가요?"
"[A] 안 되고, [B] 안 되고, 이래서 나왔습니다"

사례:
"아직도 연예인이 광고하는 가격 거품 가득한 브랜드만 고집하시나요?"
"그저 싼 제품만 찾고 계신가요?"
"이래서 나왔습니다!"
```

## 2. 신뢰 구축 카피 패턴 (상징계)

```
완판·판매 기록:
"출시 D일, ○○개 완판 기록"
"○○만 명이 선택한 제품"
"[플랫폼] 키워드 1위"

환불·보장: [판매자특성_선택 배열에 환불 보장 항목이 있는 경우에만 활용]
"출시하자마자 환불 0건이 우리의 자신감입니다"
"100% 환불 보장 — 우리 제품이 아니면 필요 없습니다"

후기 진정성:
"나도 살려고 새로 출시하여 후기수는 적지만 조작 없는 진짜 후기만 있는 M 브랜드입니다"
"소비자의 만족으로만 광고하겠습니다"

광고비 없음:
"100원도 광고비 사용하지 않았습니다"
"모델비가 이 제품 값에 50%가 넘습니다 — 저희는 그러지 않겠습니다"
```

## 3. 소거법·포지셔닝 공격 카피

```
위 공격 (비싼 브랜드):
"아직도 연예인이 광고하는 가격 거품 가득한 브랜드만 고집하시나요?"
"모델비가 이 제품 값에 50%가 넘습니다"
"이름값을 여러분이 내고 계신 겁니다"

아래 공격 (저가 제품):
"그저 싼 제품만 찾고 계신가요?"
"중국산, AS 불가, 안전 증명서 없음"
"저가형은 1년 안에 잔고장이 옵니다"

소거법 교육적 진입:
"저희 제품을 구매하지 않아도 좋습니다. 하지만 제대로 된 [카테고리]를 사기 위해선 알고 계셔야 합니다."
"시장에 올바르지 못한 관행을 뒤로한 채, 저희는 오로지 제품에만 집중했습니다."

포지셔닝 중간 진입:
"이래서 나왔습니다!"
"[브랜드명] — [핵심 가치] 하나로 승부합니다"
"소비자를 기만하는 업체들을 저는 응징하겠습니다!"
```

## 4. 기능계·경험재 카피

```

"[경쟁 방식]의 단점: [데이터]. [우리 방식]의 장점: [데이터]."
"[테스트 결과] — 직접 검증했습니다"

경험재 (감성·체험 중심):
"써본 사람들은 압니다"
"도대체 왜 계속 사는 걸까요?"
"한번 써본 분들은 꼭 재구매하시더라고요"
GIF 활용: 청소기 흡입력, 찜질기 온도, 손톱 전후 등 메리트를 직접 보여줌

후기 섹션 시작:
"도대체 왜 계속 사는 걸까요?"
"이 후기를 먼저 읽어보세요"
"3번째 구매하신 분의 솔직한 후기입니다"
```

## 5. 스토리텔링 카피 패턴

```
화자 개입:
"저는 [자신의 경험/처지]를 [기간]간 겪으며 생각했습니다."
"그들은 한 번도 [이용자 처지]를 겪어보지 않은 것 같았습니다."
"정말 [아주 작은 차이]만 달랐어도, [개선 결과]였을 텐데."

시장 고발:
"어차피 100% 만족하는 제품도 없고, 100% 만족하는 시장도 없습니다."
"세상의 모든 제품이 좋은 척합니다."

진정성·약점 인정:
"저는 광고를 많이 하지도 않고, 유명하지도 않습니다."
"쉽지도 않고, 그럴 자신도 없습니다."
"하지만 [핵심 가치] 하나만은 자신 있습니다."

사회적 연결·마무리:
"3개 팔릴 때마다 1개를 [타겟 관련 수혜자]께 기부합니다."
"이 문화를 바꿔 나가려고 합니다."
```

## 7. 카피 작성 어투 원칙

### 구어체로 쓰기 ("쓰는 것"이 아닌 "뱉는 것")
```
❌ "타고난 체형이라 변하기 힘들겠죠?"
✅ "몸매는 원래 타고나는 거 아니야?"

❌ "업소에서 사용하기에 적합한 청소기를 원하신다면"
✅ "업소에서 쓰는 청소기, 이거 하나면 됩니다"
```

### 조사 생략 기준 (구어체라도 전부 생략하면 안 된다)
```
✅ 생략해도 자연스러운 조사:
  - 목적격 '을/를': "TV 보고", "유튜브 보고", "현관 열고"
  - 주격 '이/가': "손주 왔는데", "할머니 좋아하는 거"
  → 구어체 카피에서 이 생략은 오히려 리듬감을 살린다.

❌ 생략하면 어색하거나 문법 오류가 나는 경우:
  - 처소격 '에/에서': "현관에 들어서기 전에" → "현관 들어서기 전에" ❌
  - 간접의문문 어미 '-는지': "누가 더 잘하는지는" → "누가 더 잘하냐는" ❌
    (간접의문문은 반드시 '-는지/-ㄴ지' 형태. '-냐는'은 인용절이라 의미가 달라짐)
  - 맥락 동사 오류: "누가 더 잘하냐는 열어봐야 알아요" (이 제품에서 '열어보다'가 맥락에 없으면 어색)
    → 제품·상황에 실제로 맞는 동사로: "해봐야 알아요" / "써봐야 알아요" / "직접 해봐야 알죠"
```

### 엔터 활용
```
의미 단위로 엔터.
모바일·PC 환경 모두 고려.
"하지만" 같은 단어 하나로도 문단 리드 가능.

예)
첫 번째는 흡입력입니다.

하지만

두 번째가 더 중요합니다.
```

### 질문형 활용
```
독자의 머리에 바로 들어오게 하는 방법.

"아직도 연예인 모델 광고 제품 쓰고 계세요?"
"왜 우리는 비싼 제품만 사야 할까요?"
"이거 아세요? 이 방법 쓰면 비용이 절반입니다"
```

## 8. 금지 표현 목록

```
❌ 스킵존 카피 (접촉 지점 가까운 타겟에게):
"이 제품에 관심 있으신가요?"
"오랫동안 사용할 수 있는 ○○를 원하시나요?"
"지치셨나요?" (키워드 검색해서 들어온 사람에게)
"비문증이란" / "○○이란" (이미 아는 정보 설명)
"이런 분들께 추천합니다" (접촉 지점 이미 가까울 때)

❌ 뻔한 패턴:
"특별 할인 중입니다!"
"지금 구매하시면 추가 혜택!"
"고객님만을 위한 특별가!"
"다양한 용도로 사용 가능합니다"

❌ 광범위 타겟:
"모든 분들에게 좋습니다"
"누구나 사용할 수 있어요"
"전연령 사용 가능"

❌ 화자 혼용:
"저는 이 제품을 만들었는데요... 저희는 열심히 하겠습니다"
('저' → '저희' 혼용)

❌ 과잉 장점 나열:
"이 제품이 좋은 이유 10가지"
(3개 이상부터 희석됨, 최대 5개)

❌ 비판만 하고 해소 없음:
"[경쟁사]는 이런 문제가 있습니다" → 우리 해결책 누락
```
