# 05 리스팅 제목(상품명) / 브랜드명 — 작업 지침

> 05 채널 단계의 보조 작업. 채널 분석은 **05 본체**가 담당하고, 이 작업은 상품명(채널 리스팅 제목)과 (선택) 브랜드명을 **별도 호출**로 생성한다.
> 같은 페이지에 있지만 LLM 호출은 분리되어 05 채널 분석 품질에 영향을 주지 않는다.

---

## 이 작업의 본질

- **상품명**: 온라인몰(스마트스토어/쿠팡/11번가/카페24 등)에 리스팅될 **상품 페이지 제목**. 검색 SEO와 클릭률을 좌우.
  - 제품명과 다름. 제품명은 키미테·하나로처럼 제품의 고유 이름(상표), 상품명은 리스팅 제목.
  - 채널별로 SEO 가이드라인이 다르므로 **채널을 명시**해야 함.
- **브랜드명** (선택): 회사·라인의 우산 이름. 여러 제품을 묶는 상위 정체성. 다이슨·무신사 패턴.

---

## 입력 자료

다음이 user input으로 들어온다:
- 제품 정보 (스펙)
- 선택된 타겟 (01)
- 02 포지셔닝 결과 (CV·욕구깡패·구매편익)
- 03 제품명 (이미 확정된 제품 고유명)
- 05 채널 결과 (추천 채널 순위·허브 키워드·채널별 세부 전략)
- **호출 옵션**: 브랜드명 같이 생성 여부 (true/false)
- **호출 옵션**: 채널 명시 (예: "스마트스토어", "쿠팡", "전체 추천 채널")

---

## 작업 절차

### 1) 상품명 생성 (필수)

채널이 명시되어 있으면 그 채널의 SEO 가이드 우선 적용. "전체 추천 채널"이면 05 결과의 상위 1~2개 채널별로 각각 후보 생성.

채널별 SEO 가이드:
- **스마트스토어**: 키워드 나열형. 50자 이내. [브랜드]+[제품명]+[핵심 속성]+[타겟 키워드] 순. 특수문자 자제.
- **쿠팡**: 카테고리 키워드 + 후기 유발 어휘. 100자 이내. 중복 키워드 패널티 주의.
- **11번가**: 검색량 높은 키워드 앞쪽 배치. 가격대·할인 키워드 활용 가능.
- **카페24**: 자유도 높음. 브랜딩 + 감성 어휘 가능. SEO보다 클릭률 우선.

각 채널마다 3~5개 후보 + 각 후보의 글자수·핵심 키워드 명시.

### 2) 브랜드명 생성 (옵션, 호출 시에만)

호출 시 `브랜드명_생성=true`일 때만 수행. false면 이 절은 통째로 건너뜀.

생성 시:
- 시리즈 확장 가능성 검토 (단일 제품 X) — 확장 예 2~3개 같이 제시
- 글로벌 확장 가능성 있는 제품: 영문 발음·뜻 검토
- KIPRIS (kipris.or.kr) 동일·유사 상표 검색 권고 표기
- 다이슨·무신사·삼다수처럼 짧고 입에 달라붙는 패턴 우선

3~5개 후보 + 각 후보의 시리즈 확장 예시.

### 3) 출력

```
## 📝 상품명 (리스팅 제목) — [채널명]

| 후보 | 글자수 | 핵심 키워드 | 비고 |
|---|---|---|---|
| ... | ... | ... | ... |

## 🏷️ 브랜드명 (옵션 ON 시에만)

| 후보 | 시리즈 확장 예 | 영문 표기 | KIPRIS |
|---|---|---|---|
| ... | ... | ... | 미확인 |
```

마지막에 다음 한 줄:
```
"상품명·브랜드명 검토 후 KIPRIS(브랜드명 한정)·네이버 검색량 확인 권고."
```

---

## 작업 원칙

- 상품명은 채널 SEO 가이드에 따라 가공. "홍보합니다"식 광고 문구 X.
- 03 제품명은 그대로 활용하되 채널 SEO 규격에 맞게 조합.
- 브랜드명은 옵션 OFF이면 절대 생성 X.
- 02 포지셔닝의 CV·욕구깡패 핵심 언어를 상품명 키워드로 활용.
