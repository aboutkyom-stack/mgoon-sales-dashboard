# 채널·물길 M군 검수 체크리스트 (05_channel)

> **로드 시점**: 채널·물길 전략 결과물이 나온 직후 검수 단계에서만.
> 작업 시작 시 로드하지 않는다 (토큰 낭비).
> **성격**: 판단 여지 없는 확정 규칙만. 재발 가능성 있는 실수 패턴 기록.

---

## 확정 규칙

### R-01. 설득 도구 선행 확인
채널 물길 전략이 나오기 전에 "설득 도구(상세페이지)가 준비됐는가?"가 먼저 확인되어야 한다.

- **위반 예**: 상세페이지 없이 "카페 내부 물길 → 지식인 → 블로그 순으로 시작하세요" 권고
- **통과 예**: "먼저 상세페이지를 완성한 뒤 물길을 작동시키세요. 설득 도구 없이 유입만 늘리면 전환 없음."
- **근거**: 물길의 핵심 공식 — 설득하기가 먼저, 모으기는 나중.

### R-02. 접촉 지점 거리 명시
채널 전략에 접촉 지점이 가까운지 먼지 명시되어야 한다.

- **위반 예**: "블로그로 타겟을 모으세요" (접촉 지점 거리 판단 없음)
- **통과 예**: "이 제품은 접촉 지점이 가까운 타겟 대상 → '판매 1위' 강조 방향 / 먼 타겟 대상 → 새로운 기준 제시 방향"
- **근거**: 거리에 따라 오프닝·카피 전략이 달라진다.

### R-03. 카페 내부 물길 리얼리티 조건
카페 내부 물길 전략 제안 시 반드시 "활동성 쌓기 선행 + 제품명 직접 언급 금지" 조건이 포함되어야 한다.

- **위반 예**: "신혼부부 카페에 로봇 청소기 후기 올리세요"
- **통과 예**: "신혼부부 카페 가입 후 2~4주 활동성 쌓기 → 문제 호소 글 → 댓글/쪽지로 간접 연결. 본문에 제품명 직접 언급 금지."
- **근거**: 카페 내부에서 홍보티 노출 시 즉시 퇴출.

### R-04. 정복 키워드 직접 공략 금지
5단계 정복 키워드(광범위 정보성 키워드)를 바로 공략하라는 권고는 금지.

- **위반 예**: "다이어트" 키워드 블로그 작성 권고
- **통과 예**: "'다이어트 식품 비교', 'OO 성분 효능' 등 3~4단계 세분화 키워드 공략"
- **근거**: 정복 키워드는 대형 미디어·파워 블로거가 점령 — 신규 진입 불가.

### R-05. 지식인 IP 분리 조건 명시
지식인 전략 제안 시 IP 분리 방법 언급이 포함되어야 한다.

- **위반 예**: "질문 아이디와 답변 아이디를 각각 만드세요"
- **통과 예**: "질문·답변 아이디 각각 준비 + 동일 IP 사용 금지 (휴대폰 비행기 모드로 IP 변경)"
- **근거**: 동일 IP 사용 시 어뷰징 감지 → 계정 정지.

### R-06. 허브 키워드 의심·독식 조건 확인
허브 키워드 제안 시 "의심받지 않는 정보성 + 독식 가능성" 두 조건 모두 확인해야 한다.

- **위반 예**: "장어즙" 키워드 직접 공략 (이미 제품 키워드)
- **통과 예**: "발기 효능 식품" (정보성, 경쟁 약함, 의심 없음) → 장어즙 자연 연결
- **근거**: 허브 키워드는 타겟이 스스로 탐색하는 키워드여야 거부감 없이 작동한다.

### R-07. 쿠팡 후기 물길 — 보완재 조건
쿠팡 후기 물길 제안 시 보완재(complementary product)가 동일 카테고리 경쟁 제품이 아닌지 확인.

- **위반 예**: 청소기 후기에 "이것보다 OO 청소기가 더 좋아요" 형태 추천
- **통과 예**: 성장 영양제 후기 → "이 제품과 함께 거꾸리 성장 기구 쓰면 효과 2배" (보완재)
- **근거**: 경쟁 제품 추천은 자사 제품 신뢰 파괴 + 쿠팡 정책 위반 소지.

### R-08. 스마트스토어 후기 목표 수 명시
스마트스토어 전략 제안 시 후기 수 기준(5,000개/30,000개)이 언급되어야 한다.

- **위반 예**: "후기를 많이 쌓으세요"
- **통과 예**: "초반 목표: 5,000개 (판매 수월해짐) → 장기 목표: 30,000개 (수익 안정화). 초반엔 우편 발송 가능한 사은품으로 후기 확보."
- **근거**: 목표 수 없이는 방향이 없다. 30,000개는 장기전 마인드 세팅에 필수.

## 항목 추가 기준

다음 **모두** 만족할 때만 여기 추가:
- 동일/유사 실수가 **2회 이상** 발생
- **다른 제품·다른 세션에서 재발 가능**한 표면 실수
- **판단 여지 없이 규칙으로 확정 가능** (맥락 판단이 필요하면 별도 메모)
