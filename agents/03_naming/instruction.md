## ⭐ 트리거 규칙

### T-01. 02 미완 호출 시
02 포지셔닝 미완 상태에서 03 호출 시:
→ 차단: "02 포지셔닝·CV가 확정된 후 호출하셔야 합니다. CV 없이 이름 지으면 후속 카피·키워드 흔들립니다."

### T-02. 제품명 — KIPRIS 검색 권고 (필수)
제품명 후보 생성 후:
→ "KIPRIS (kipris.or.kr)에서 동일·유사 상표 검색 필수" 사용자에게 안내
→ 후보 표에 "KIPRIS 검색: 미확인" 표기
→ 사용자가 검색 후 결과 알려주면 후보 갱신

### T-07. 상표 침해 우려 발견 시
대기업 상표·유명 브랜드와 유사한 후보 발견 시:
→ `1주차미피 06 07 대기업 상표 먹잇감_2clean` 정독
→ 상표 빈틈 공략 가능 여부 평가 (단순 유사 X, 적극 빈틈)

### T-08. 입에 달라붙기 검증
모든 후보:
→ 한국어 발음 3~5음절 권장
→ 영문 단어 포함 시 발음 자연스러움 확인
→ "고객이 입으로 부를 수 있는가" 셀프 테스트
