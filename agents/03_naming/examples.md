# 네이밍 M군 — 실전 예시 (03_naming)

> 강의 `2주차 06 네이밍만 잘 해도 반은 먹고 들어갑니다` (_2clean)에서 추출.
> 강사 실제 멘트는 큰따옴표 안에 원문 그대로 보존.

---

## 성공 사례 분석

### 1. 키미테 (귀 밑에 붙이는 멀미약 패치)

**네이밍 전략**: 위치·부위 직결형
**타겟**: 멀미 심한 사람, 특히 어디 붙이는지 모르는 사람

```
기존 경쟁: 멀미약 = 먹는 약 (동해드라민 등)
키미테 전략: "귀(耳) 밑(下)" → "키-미-테" → 위치가 이름 자체
```

강사 멘트:
> "키미테는 귀 밑에 붙이자, 너무 똑똑한 위치로 한 거예요.
> '코 밑에 키미테!'라고 광고해도 재밌는 거죠. 저는 역사상 최고의 네이밍이 아닌가."

**왜 성공했나**:
- 타겟의 핵심 궁금증("이거 어디 붙이는 거야?")을 이름 자체가 해결
- 관여도 낮춤 효과 — 어디 붙이는지 설명할 필요가 없어짐
- 브랜드명 = 사용 설명서

---

### 2. 하나로 샴푸 (샴푸+린스 일체형)

**네이밍 전략**: 핵심 가치(CV) 직결형
**타겟**: 샴푸와 린스를 따로 사는 것이 번거로운 사람

```
시장 상황: 랑데뷰 샴푸, 투웨이 샴푸 등이 시장 리더
당시 트렌드: 샴푸+린스를 하나로 합치는 기술이 가능해진 시점
후발주자 '하나로'의 전략: 기능명 = 제품명
```

강사 멘트:
> "'하나로'가 네이밍만으로 그냥 끝낸 거죠.
> 네이밍만으로 시장 1위가 된 아주 이례적인 케이스예요.
> 시장의 후발주자였어요. 근데 이름 하나로."

**왜 성공했나**:
- 이름 자체가 핵심 가치(편리함, 일체화)를 설명
- 경쟁사가 "감성·예쁜 이름"(랑데뷰, 투웨이)을 쓸 때 혼자 기능을 이름으로 씀
- 구매 결정 순간 "아, 이게 하나로 되는 거구나"가 이름에서 즉시 해결됨

---

### 3. 배달의민족

**네이밍 전략**: 가치 직결형 + 문화적 공감
**타겟**: 배달 음식 주문하는 한국인 전체

```
서비스: 음식 배달 중개 앱
포지셔닝: "배달을 잘 하는 민족(한국인)이 만든 배달 앱"
회사명: 우아한형제들 (유형 1 — 의미 중심)
브랜드명: 배달의민족 (유형 2 — 포지셔닝 중심)
```

강사 멘트:
> "배달의민족이었어야 된다고 생각해요, 회사명도. 처음에 그런 생각을 못 했었으니까."

**왜 성공했나**:
- 앱이 무엇을 하는지 이름만으로 100% 전달
- "민족"이라는 단어로 한국인 정체성과 공감대 형성
- 유머가 섞인 이름이라 기억에 남고 언급되기 쉬움

---

### 4. 돼지코팩

**네이밍 전략**: 가치 더하기형 (기능 변경 없음)
**타겟**: 코 모공 관리에 관심 있는 소비자

```
기존 시장: 코팩 (일반 명칭)
돼지코팩 전략: 네이밍 + 패키지 디자인만 교체 (기능은 동일)
결과: 시장 장악
```

강사 멘트:
> "우리의 모든 네이밍은 이 '돼지코팩'처럼 돼야 돼요.
> 네이밍만으로 기존에 똑같은 기능인데 다른 것처럼 보여서 시장을 먹어버리잖아.
> 이거는 사실 네이밍, 패키지, 디자인이 다 한 거죠.
> 실제 기능이 안 바뀌었기 때문에 '가치 더하기'라고 합니다."

**왜 성공했나**:
- 코팩의 용도(코 모공) + 귀여운 시각화(돼지코) 조합
- 사진 없이 이름만으로 사용 장면이 연상됨
- 기억에 남고 공유·언급하기 쉬운 이름 → 자연 바이럴

---

### 5. 탑블로 (등산용 텀블러) — 강의 중 실시간 집단지성 사례

**네이밍 전략**: 위치·상황형 + 가치 직결형
**타겟**: 등산하는 사람

```
타겟 세분화 과정:
텀블러 (광범위) → 직장인도 있고, 등산도 있고, 캠핑도 있고
→ "등산용 텀블러"로 타겟 세분화

네이밍 집단지성 과정:
강사: "뭐라고 붙일까요?"
수강생: "마운틴블럭? 뭐 이런 것도 되고"
수강생: "탑블로!"
강사: "오, 이거 좋다. 집단지성은 이렇게 나온다."
```

강사 멘트:
> "탑이 이제 높은 데, 그런 거를 '하이'나 '탑' 뭐 이런 거로 대체할 수도 있는 거죠.
> 너무 좋았어요. 그렇게 너무 가벼운데, 그거 하나 차이가 되게 크다고, 시장 들어갈 때.
> '아, 이건 등산 전용이구나.' 등산 중에 중요한 기능, 남의 거 두 개 제거하고 우리가 두 개 넣고,
> 이렇게 시작하면 되는 거죠. 쉬워요."

**왜 좋은 네이밍인가**:
- "탑(정상)"이라는 등산인의 세계 언어 활용
- 일반 텀블러와 즉시 구분됨
- 등산 관련 커뮤니티·검색에서 자연스럽게 어울리는 이름

---

### 6. 눈엔 (안구건조증 점안액) + 시리즈 확장

**네이밍 전략**: 위치·부위형 + 시리즈 확장 구조
**타겟**: 눈이 건조한 사람

```
핵심 구조: [부위] + [조사 '엔']
눈 + 에 → 눈엔
시리즈 확장 가능성: 두피열엔, 무릎엔, 어깨엔, 기관지엔, 입술엔...
```

강사 멘트:
> "'엔'으로 다 넣었더라고요. 아직 제품이 안 나온 것도 있고 나온 것도 있는데.
> 이런 게 똑똑한 거죠. 내가 만약에 여섯 개만 뺏고 있었어도 몇억 벌지 않았을까요?
> 걔들이 그거를 사야 되니까."

**왜 좋은가**:
- 단순하고 직관적 (부위 + 조사)
- 시리즈 확장이 쉬운 구조 → IP·브랜드 자산 선점
- 이름에서 사용 위치가 즉시 연상됨

---

### 7. 가그린, 아이셔, 달콤

**공통 전략**: 기능·감각·포지셔닝이 이름에 녹아있는 유형 2

```
가그린: 가글 + 그린(상쾌함) → 구강청결제
아이셔: '아이, 셔!'라는 감탄사 → 신맛 사탕의 반응이 이름
달콤: 맛 자체가 이름 → 디저트 연상
```

강사 멘트:
> "'아이셔'는 자기가 아이셔, 시기 때문에 기능적으로 한 거지만,
> 사람들한테 포지셔닝이 '아이, 셔!'로 되는 거잖아요.
> '달콤'도 굉장히 좋은 예시죠. 새우깡, 굽네치킨, 다나와 다 마찬가지입니다."

## 실패 사례 분석

### 1. '치킨 철판이야' (강사 실제 운영 가게)

**원인**: 판매자 언어를 이름으로 씀, 고객 언어 무시

```
판매자가 지은 이름: "치킨 철판이야" (이유: 철판에 나왔으니까)
고객들이 실제로 부른 이름: "슬러시 맥주집"
결과: 두 이름이 충돌 — "슬러시 맥주집"이 이김
```

강사 멘트:
> "저희 가게는 '치킨 철판이야'라는 이름이 있었습니다, 제가 했던 가게는.
> 근데 이제 가짜 철판이라 사람들이 이렇게 생각을 안 하더라고.
> 사람들이 이제 앞에 전화하는데 '야, 슬러시 맥주집으로 와.' 이러더라고.
> 나는 이쪽으로 정해서 썼어야 됐다. 슬러시 맥주로 썼어야 됐다."

**교훈**:
- 고객이 이미 붙인 별명이 진짜 이름
- 판매자가 강조하고 싶은 것(철판 조리법)은 고객에게 의미 없음
- 고객의 기억 속에 뭐로 남는지가 네이밍의 답

---

### 2. Way Back Home (노래 제목)

**원인**: 고객이 기억하는 가사가 제목보다 더 강력

```
공식 제목: Way Back Home (영어, 입에 달라붙지 않음)
고객들이 기억하는 구절: "멈춘 시간 속에"
검색: "Way Back Home"이 아닌 "멈춘 시간 속"으로 찾아야 나옴
```

강사 멘트:
> "'멈춘 시간 속에', 기억이 잘 남아요? 'Way Back Home', 기억이 잘 남아요?
> '멈춘 시간 속'이 제목이었으면 잘 찾았겠지. 이게 좋은 네이밍인 거죠.
> 이건 네이밍이 잘 됐나요? 안 된 거지. 'Way Back Home'이 입에 안 달라붙는 거지."

**교훈**:
- 아무리 좋은 제목이어도 "입에 잘 달라붙지 않으면" 실패
- 고객이 기억하는 구절, 고객 언어가 곧 네이밍 소재
- 블로그 제목에도 동일 적용: 클릭률 높은 표현 = 사람들이 이미 쓰는 표현

---

### 3. '캐나다 삼촌네' (제주도 분식집)

**원인**: 기억에 남지만 무엇인지 전달 안 됨

```
강사 경험: 제주도 여행 중 '캐나다 삼촌네' 이름에 끌려 방문
실제: 손님이 없고, 가정집 느낌, 기대와 다름
평가: 이름의 독특함 ≠ 내용물 전달
```

강사 멘트:
> "이름이 뭐였냐, '캐나다 삼촌네'. 어, 되게 줄 설 것 같지 않아요?
> 가서 보니까 우리밖에 없었어.
> 뭐 동네 정보가 없을 땐 네이밍이 더 중요하고요."

**교훈**:
- 유형 1(의미 중심) 이름은 브랜드 스토리가 뒷받침되어야 함
- 이름이 독특해도 고객이 "무엇인지" 모르면 재방문이 없음
- 신규 브랜드에게 유형 1이 위험한 이유

## 블로그 제목 네이밍 실전 예시

### 결핍 자극 + 검색어 조합 패턴

```
[검색 키워드] + [결핍 자극]
- 층간소음 귀마개 추천 — 잠 못 자는 분들 이거 보세요 (야간근무자 후기)
- 멀미 패치 어디 붙이는지 모르면 효과 없어요 (키미테 정확한 위치)
- 두피열 샴푸 — 3개월 써봤습니다 (미용실 원장도 추천한 방법)
```

### 타겟 직접 지목 패턴

```
[타겟 상황 직접 묘사] → 클릭률 상승
- 야간근무 후 잠 못 드는 분들 보세요
- 갱년기 엄마 선물 고민이세요? 이거 하나면 됩니다
- 생리통 매번 진통제 드시는 분들, 이걸 몰랐던 거예요
```

### 숫자 + 결핍 패턴

```
[구체적 숫자] + [문제 해결/결과]
- 3만 원 이하로 1주일 층간소음 해결한 방법
- 딱 2가지만 바꿨더니 수면의 질이 달라졌어요
- 10년 무릎 통증, 이걸로 3일 만에 잡았습니다
```

## 스마트스토어 상품명 네이밍 실전 예시

### 타겟 + 상황 + 제품 조합 공식

```
[타겟 상황] + [핵심 기능] + [제품 카테고리]

- 층간소음 귀마개 취침용 실리콘 귀마개 소음차단
- 야간근무 수면 귀마개 피로회복 불면증 개선
- 갱년기 어머니 선물 온열찜질기 복부 허리 생리통
- 생리통 심한 여성 전용 배찜질기 따뜻한 복부 온열
```

> **원칙**: 검색 키워드 + 타겟 언어 + 결과/효과를 조합.
> 상품명 = 검색 잘 되는 문장이자, 타겟이 "이건 내 거다"를 느끼는 문장.

## 실전 체크리스트

네이밍 완성 전 반드시 확인:

- [ ] 고객이 이 제품·서비스를 이미 어떻게 부르는지 조사했는가?
- [ ] 유형 2 (포지셔닝 중심)를 우선 시도했는가?
- [ ] 이름을 처음 듣는 사람이 3초 안에 무엇인지 연상되는가?
- [ ] 타겟의 언어(검색어, 커뮤니티 표현)가 반영되어 있는가?
- [ ] 경쟁사가 이미 선점한 언어와 차별화되는가?
- [ ] 블로그 제목이라면 검색 키워드가 앞 10자 이내에 있는가?
- [ ] 한국어로 발음하기 쉽고 3~5음절 이내인가? (제품명·브랜드명 한정)
- [ ] 시리즈 확장 가능한 구조인가? (선택)
