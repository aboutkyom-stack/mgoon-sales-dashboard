## ⭐ 트리거 규칙

### T-01. 5단계 정복 키워드 권고 시 (필수 차단)
권고한 키워드가 "다이어트", "수면" 같은 5단계 정복 키워드면:
→ **즉시 재분석**. 정복 키워드 직접 공략 금지
→ 3~4단계 세분화 키워드로 재제안
→ 4주차2 07 _2clean 재확인

### T-02. 허브 키워드 선정 시 (필수)
허브 키워드 결정 시:
→ `4주차2 08 나보다 싸게 파는 판매자 허브키워드_2clean` 정독
→ 독점 가능성 평가 (검색량 + 경쟁 강도 + 우리 진입 우위)

### T-03. 카페 진입 권고 시
타겟 카페 1~3개 선정 시:
→ `4주차1 09 카페에서 물길_2clean` 정독
→ 카페 운영 규칙·광고 차단 정책 확인
→ 직접 광고 X. 정보형 글 + 시그니처

### T-04. 블로그 외주 권고 시
글 작성을 외주로 권고하면:
→ `4주차1 08 블로그3 외주 줄 때_2clean` 정독
→ 키워드·제목·소제목·핵심 메시지는 내가 직접 (외주 X)

### T-05. 스마트스토어 노출 부족 진단 시
구매량·클릭률·활동성 중 어느 것이 부족한지 진단 + 강화 방안:
→ `4주차2 10 스마트스토어 노출 3대장_2clean` 정독
→ 셋 다 동시 강화 방안 (광고비만으로는 한계)

### T-06. 대체재 전략 권고 시
"우리는 ○○의 대체재" 포지션 권고 시:
→ `4주차2 06 물길 모으고 설득하기 대체재_2clean` 정독
→ 50% 가격 / 100% 욕구 충족 검증

### T-07. 경쟁사 견제·신고 우려 발견 시
운영 중 업체 신고·경쟁사 견제 우려:
→ `2주차미피 01 업체 신고 경쟁사 견제_2clean` 정독
→ 합법적 대응 방안 + 차단 우회

### T-08. 지역 키워드 권고 시
지역 키워드 활용 권고 시:
→ `4주차2 07 2주만에 2천 지역 키워드_2clean` 정독
→ 지역별 검색량·경쟁도 매핑

### T-12. 상세페이지 미완 의심 시
05 작업 시작 전에 04 결과(`items/item_<id>/04_a_writing.md` + `04_b_review.md`)가 없거나 부실하면:
→ **05 진행 정지**. 사용자에게 "설득 도구가 먼저"라고 안내
→ 04 완료 후 05 재개
