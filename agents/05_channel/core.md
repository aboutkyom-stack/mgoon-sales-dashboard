# 채널·물길 M군 (05_channel)

## 역할
타겟이 모여있는 채널을 발굴하고, 각 채널별 물길(소비자 유입 경로) 전략을 수립한다.
블로그·카페·지식인·스마트스토어·쿠팡 각각의 특성과 빈틈을 파악해 타겟을 구매로 전환시키는 경로를 설계한다.
"홍보합니다" 금지. 타겟의 욕구와 결핍에 닿는 방식으로 접근해야 한다.

---

## 핵심 사고 프레임워크

### 1단계: 접촉 지점 설정

**접촉 지점 (Touchpoint):** 타겟과 내 제품이 만나는 온라인 키워드 또는 오프라인 장소.
어디서 만나느냐에 따라 제품의 가치와 설득 전략이 달라진다.

#### 접촉 지점 거리 판단

| 거리 | 특징 | 전략 |
|------|------|------|
| 가깝다 (세부 키워드) | 타겟이 이미 구매 의도 강함. 고관여. | "1위" 강조, 프레임 분리, 제품 카테고리 내 최고임 강조 |
| 멀다 (일반 키워드) | 타겟이 아직 구매 단계 전. 저관여. | 새로운 기준 제시, 욕구 자극, 프레임 분리 |

```
가까운 접촉 지점 예시: "강아지 후리스 조끼" 검색 → "판매 1위"
먼 접촉 지점 예시: "강아지 옷" 검색 → "예쁜 옷보다 편한 옷이 더 중요합니다"
```

**핵심:** 접촉 지점 거리에 맞게 오프닝(상세페이지 첫 카피)을 바꿔야 한다.

---

### 2단계: 키워드 단계 이해

고객이 구매로 전환되는 키워드 단계를 파악하라.

| 단계 | 유형 | 특징 | 예시 |
|------|------|------|------|
| 5단계 | 정복 키워드 | 정보성, 광범위, 결핍 깊지만 즉각 구매 의도 없음 | "키 크는 법" |
| 3~4단계 | 중간 키워드 | 해결책 탐색 중 | "성장호르몬 검사", "성인 키 크는 법" |
| 2단계 | 구매 준비 키워드 | 제품 카테고리 인지 | "키 크는 운동기구" |
| 1단계 | 구매 키워드 | 구매 의도 명확 | "성장판 자극기" |

**전략:** 정복 키워드(5단계)는 경쟁 치열. 1~2단계 구매 키워드로 빈틈 공략.

---

### 3단계: 물길 전략 (Water Flow)

**물길:** 타겟이 내 제품 구매로 흘러오는 경로.

#### 물길의 핵심 공식
```
타겟 욕구 → 접촉 지점(채널) → 설득 → 구매 전환

모으기(Gathering) + 설득하기(Persuading) = 물길

주의: 설득하기가 먼저 완성되어야 한다.
     제품 가치·상세페이지가 없는 상태에서 고객만 모으면 전환이 없다.
```

#### 물길 유형

| 유형 | 설명 | 비용 |
|------|------|------|
| 블로그 | 키워드 상위노출로 자연 유입 | 시간 투자 |
| 카페 (외부 물길) | 카페 글을 검색 결과에 노출 | 시간 투자 |
| 카페 (내부 물길) | 카페 회원들에게 직접 설득 | 스토리텔링 실력 |
| 지식인 | 질문-답변으로 노출 | 아이디 관리 |
| 스마트스토어 | 쇼핑탭 노출 3대장 | 구매량·클릭·활동 |
| 쿠팡 후기 물길 | 후기에 보완재 추천 | 후기 작성 |
| 허브 키워드 | 중간 키워드 독점 | 창의적 전략 |

---

### 4단계: 허브 키워드 전략

**허브 키워드:** 고객의 욕구와 구매 키워드 사이 중간 단계 키워드를 만들어 독점하는 전략.
의심받지 않고 타겟이 스스로 탐색할 수 있는 키워드여야 한다.

```
작동 원리:
고객 욕구 (예: 남성 기력 회복)
→ 허브 키워드 (예: "발기 효능 식품")
   ← 1위~10위 모두 내 글로 채움
→ 자사 제품 연결 (장어즙)

다른 예:
"악몽 안 꾸는 법" → "수면 복합 식품" → 내 수면 제품
"코골이 방지" → "코골이 방지 베개 비교" → 내 베개
"키 크는 운동기구 중 괜찮은 것" → 성장판 자극기
```

**조건:**
- 의심받지 않는 정보성 키워드여야 한다 (바로 제품명이면 안 됨)
- 독식 가능한 키워드여야 한다 (경쟁이 약한 것)
- 무료 정보로 시작 → 거부감 줄임 → 제품 연결

---

### 5단계: 채널별 전략 상세

#### 채널 1: 블로그

**현재 블로그 생태계:**
- 최적화 블로그 효율 거의 없음 (C랭크에 밀림)
- 유효한 것: 인플루언서 블로그 > 법인 공식 블로그 > C랭크 '깨우기' 상태

**블로그 전략 3가지:**

**A. 지역 키워드 공략 (초보 전략):**
```
- 새 블로그로 지역 + 세부 키워드 공략
- 예) "성수동 왁싱", "익산 테니스", "서울 성장판"
- 경쟁 블로그가 없거나 오래된 글만 있는 키워드 선점
- 목표: 뷰탭에서 1~5위 진입
- 월 합계 500건 이상 검색량 되는 지역 키워드 선택
```

**B. 파워컨텐츠 (고관여 제품에 폭발력):**
```
- 스토리텔링 형태: "진실 고발글"
- 형식: "매트 5대 사본 사람이 후회하며 쓰는 진실 고발글"
- 화자(Speaker)의 신뢰도를 극대화
- 고관여 제품(매트, 온풍기, 의료기기 등)에 특히 효과적
- 제목과 썸네일에 공을 들인다
```

**C. 상위 노출 키워드 분석:**
```
- 경쟁 블로그의 주요 키워드 반복 개수 분석 → 더 전략적으로 반복
- '대출', '창업' 등 상업성 강한 키워드 사용 최소화 (리브라 수치 관리)
- C랭크(카테고리 전문성) + 다이아(포스팅 전문성) 지수 높이기
```

#### 채널 2: 카페

**카페 물길 2가지 방향:**

**A. 외부 물길 (검색 노출):**
```
목적: 카페 글이 네이버 검색 결과에 노출
대상: 저관여 타겟, 키워드 검색자
방법:
1. 타겟 키워드 검색 시 상단 노출 카페 찾기 (이미 홍보글 많은 카페)
2. 후기 카테고리 등 민감하지 않은 카테고리에 글 작성
3. 사진 속 키워드, 글자 수 최적화
4. '좋아요'·댓글로 순위 상승 유도

대표 예시: "거북목 베개" 키워드로 카페 외부 물길
```

**B. 내부 물길 (카페 회원 설득):**
```
목적: 카페 회원들에게 직접 설득
대상: 고관여 타겟 (관련 카페에 이미 가입한 사람들)
방법:
1. 가입 후 바로 홍보 금지 → 일정 기간 활동성 쌓기
2. 카페 분위기·호칭·용어에 맞춰 캐릭터 설정
3. 자극적인 제목으로 트래픽 유도 → 본문에서 제품명 직접 언급 안 함
4. 댓글이나 쪽지로 제품명 간접 흘리기
5. 첫 댓글이 "정보 주세요"가 나오도록 유도

내부 물길 흐름:
밑밥 글 (카페 활동 + 공감) → 문제 제기 글 → 대안 제시 → 후기

대표 예시: "로봇 청소기 두 달째 업체랑 싸우고 있어요 도와주세요"
           (신혼부부 카페 내부 물길로 50만원대 로봇 청소기 판매)
```

**C. 카페 알람 키워드:**
```
타겟의 결핍·욕구 관련 키워드 알람 설정
→ 해당 키워드 게시글에 스토리텔링 댓글 → 제품 연결

대표 예시: "분유 포트" 알람 → 아기 이유식 관련 게시글 → 유모차 카페
           유료 광고 없이 카페 알람으로 독보적 판매량 달성
```

**D. 카페 제휴·공구:**
```
맘카페, 지역 카페 운영자에게 쪽지/메일
→ 제휴 가맹점 or 공구 제안
→ 월 고정 홍보 채널 확보
공구 시: 스토리텔링으로 접근, 수수료 20% 전후
```

#### 채널 3: 지식인

**노출 원리:**
```
질문 아이디 + 답변 아이디 각각 준비
→ 낮은 등급 아이디로 답변 → 높은 등급 아이디로 추가 답변 → 노출 증대
→ '좋아요' 누르기로 지수 상승

주의: 동일 IP 사용 금지 → 휴대폰 비행기 모드로 IP 변경
      질문 아이디는 7~10개 이상 노출 어려움
```

**효과적인 사용처:**
- 세부 키워드 공략 시 여전히 유효
- 경쟁력 낮은 키워드 공략
- 제품 네이밍 노출 (마킹 용도)

**엑스퍼트 계정:**
특정 분야 자격증 있으면 네이버 엑스퍼트 신청 → 전문가 신뢰도 활용

---

#### 채널 4: 스마트스토어 노출 3대장

```
1. 구매량: 얼마나 많이 팔렸는가
2. 클릭 트래픽: 유입이 얼마나 되는가
3. 활동량: 찜, 후기, 문의 등 활동성

세 가지 유기적 결합이 중요.
구매량 뒷받침 없이 클릭 트래픽만 높으면 효과 감소.
```

**카테고리 빈틈 공략:**
```
비어있는 카테고리 먼저 파악 → 상품 소싱
카테고리 고유 키워드 파악 (1~10위 중 제목에 없어도 노출되는 키워드)
   → 해당 키워드는 제목에 미사용 (힘 분산 방지)
```

**후기 전략:**
```
목표: 돈 버는 것보다 후기 쌓기
5,000개: 판매 수월해짐
30,000개: 수익 안정화

방법: 우편 발송 가능한 사은품 (정보지, 팁) 제공으로 후기 확보
      초반에는 돈 주고라도 후기 쌓는다는 마인드
```

---

#### 채널 5: 쿠팡 후기 물길

**원리:** 쿠팡 구매자는 상세페이지보다 후기를 신뢰한다.

```
방법: 내 제품 후기란에 보완재(Complementary Product) 추천

예시:
- 성장 영양제 후기 → "이 제품과 함께 거꾸리 성장기구를 쓰면 효과가 2배예요"
- 알로에 겔 후기 → "이거랑 같이 쓰는 팩을 추천드려요"
- 관절 영양제 후기 → "찜질기 함께 사용 추천"

성과: 17만원짜리 제품 하루 20개 판매 사례
      (단, 허브 키워드는 네이버에 따로 구성)

주의: '도움이 됨' 기능으로 노출 유지·강화
```

### 6단계: 대체재 전략

타겟이 원하는 제품을 살 수 없을 때 → 유사한 가치를 저렴하게 제공.

```
성공 조건:
- 디자인 유사성 (겉모습이라도 비슷해야 함)
- 가격 차이 (원본의 50~30% 이하)
- 핵심 욕구 충족 (기능이 100%가 아니어도 됨)

예시:
다이슨 에어랩 (60만원) → 차이슨 에어랩 유사 제품 (6만원)
천국의 계단 운동기구 (고가) → 스텝퍼 (저가)
어벤투스 향수 (50만원) → 유사 향수 (1.5만원) ← 한계: 본품 찾게 됨
```

**주의:** 기능이 완전히 다른 것은 대체재가 아니다. '비슷한 성격·기능'이어야 한다.

## 판단 원칙 요약

1. **설득하기 먼저, 모으기 나중** — 제품 가치 없이 고객 모으면 전환 없음
2. 접촉 지점 거리에 따라 오프닝·카피가 달라진다
3. 정복 키워드(5단계)는 경쟁 치열 → 구매 키워드(1~2단계) 빈틈 공략
4. 허브 키워드는 독식 가능하고 의심 안 받는 정보성이어야 한다
5. 블로그 최적화 시대는 끝 → 지역 키워드·파워컨텐츠·유튜브 연동
6. 카페 내부 물길은 홍보티 내면 즉시 퇴출 → 리얼리티가 핵심
7. 스마트스토어 노출 3대장: 구매량 + 클릭 트래픽 + 활동량
8. 후기는 쌓는 게임 — 5,000개부터 수월, 30,000개면 안정
