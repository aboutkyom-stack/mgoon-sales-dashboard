# 네이밍 M군 — 표현 뱅크 (03_naming 특화)

> 원본 `persona/mgoon/expressions.md`에서 네이밍 관련 표현 발췌 + 03_naming 작업 특화 패턴 추가.
> 이름·제목·상품명을 **실제로 생성**할 때 이 파일의 패턴을 참고한다.

---

## 1. 네이밍 패턴 유형별 표현

### 패턴 A: 위치·부위·상황 직결형

"어디에, 언제, 어떤 상황에서 쓰는 것인가"를 이름이 직접 말한다.

```
구조: [위치/상황] + [제품 카테고리 약어·조사]

사례:
- 귀 밑 + 테이프 → 키미테
- 눈 + 에 → 눈엔
- 두피 + 열 + 에 → 두피열엔
- 무릎 + 에 → 무릎엔
- 산 정상 + 텀블러 → 탑블로
- 야간 + 수면 + 이어플러그 → 야간수면이어플러그

조합 방식:
[신체부위] + 엔/에 → 해당 부위 제품
[상황] + [카테고리 핵심 단어] → 상황 특화 제품
[타겟 행동] + [카테고리] → 용도 특화 제품
```

**적용 기준**: 제품이 특정 부위·장소에 쓰이고, 그 위치 정보가 구매 결정에 영향을 줄 때.

---

### 패턴 B: 핵심 가치(CV) 직결형

제품이 주는 핵심 가치 한 단어·구절을 이름으로.

```
구조: [CV 명사] or [CV 동사화] or [핵심 행위]

사례:
- 샴푸+린스 통합 → 하나로
- 가글 + 상쾌함 → 가그린
- 신맛의 감탄사 반응 → 아이셔
- 달콤한 경험 → 달콤
- 하루 60마리 제한 → 희소성이 이름
- 배달하는 민족 → 배달의민족

조합 방식:
[결과/효과 한 단어] → 직접 제품명
[감탄·반응 표현] → 경험 연상형
[핵심 행위] + [대상] → 서비스명
```

**적용 기준**: 이 제품의 CV가 한 단어·한 구절로 명확히 말해질 때.

---

### 패턴 C: 타겟 직접 지목형

이름 안에 타겟의 상황·욕구가 들어있어 "이건 내 얘기다"를 느끼게 한다.

```
구조: [타겟 상황/욕구] + [제품]
      [타겟 직업/나이] + [용도]

블로그 제목·스마트스토어 상품명에 특히 효과적:

- 층간소음 + 귀마개 → "층간소음 귀마개 추천"
- 야간근무자 + 수면 → "야간근무 수면 전용 이어플러그"
- 갱년기 엄마 + 선물 → "갱년기 어머니 선물용 온열찜질기"
- 생리통 + 여성 → "생리통 심한 여성 전용 복부찜질기"
- 수험생 + 집중 → "수험생 집중력 향상 귀마개"
```

**적용 기준**: 검색 유입이 필요한 블로그 제목, 스마트스토어 상품명.

---

### 패턴 D: 결핍 자극형 (제목에 특화)

이름을 보는 순간 "나 이 문제 있는데!"를 느끼게 한다.

```
구조: [문제 상황 직접 묘사] / [경고형] / [의문형]

경고형:
- "이거 모르면 층간소음 절대 못 줍니다"
- "멀미약 어디 붙이는지 모르면 효과 없어요"
- "귀마개 이렇게 쓰시나요? 잘못된 방법이에요"

의문형:
- "두피열 왜 생기는지 아세요?"
- "왜 귀마개 써도 소음이 들릴까요?"
- "이거 아세요? 생리통 진통제 없이 잡는 방법"

공감 묘사형:
- "야간근무 후 잠 못 드는 분들 보세요"
- "생리통 매번 진통제 드시는 분들"
- "층간소음 때문에 이사까지 고민하셨나요?"
```

**적용 기준**: 블로그 제목, 유튜브 썸네일 텍스트. 클릭률이 목적인 콘텐츠 제목.

---

### 패턴 E: 숫자 활용형

구체적 숫자가 들어가면 신뢰와 클릭률이 높아진다.

```
구조: [숫자] + [결과/기간/방법수]

기간 + 결과:
- "2주 만에 무릎 통증 잡는 루틴"
- "3일 만에 생리통이 줄었어요"
- "1주일 써보고 솔직히 말씀드립니다"

가격 + 해결:
- "5만 원 이하로 층간소음 해결한 방법"
- "3만 원으로 수면의 질 올린 후기"

개수 + 핵심:
- "3가지만 바꿔도 수면의 질이 달라진다"
- "1개로 샴푸+린스 해결하는 방법"
- "귀마개 추천 TOP 5 (야간근무자 직접 테스트)"
```

---

### 패턴 F: 가치 더하기형 재포지셔닝

기능은 같은데 이름으로 다른 시장·타겟을 노린다.

```
기존 → 가치더하기 네이밍:
코팩 → 돼지코팩 (귀여운 비주얼 연상)
텀블러 → 탑블로 (등산 전용 연상)
샴푸 → 하나로 (편리함 강조)
귀마개 → 층간소음 수면 전용 귀마개 (타겟 세분화)

재포지셔닝 공식:
기존 일반 카테고리명 + 타겟 상황/특성 = 새로운 포지션
```

**적용 기준**: 레드오션에서 타겟 세분화나 포지셔닝 차별화가 필요할 때.

## 2. 블로그 제목 고클릭률 패턴

### 기본 구조
```
[검색 키워드] + [타겟 자극 요소]
또는
[결핍 자극] + [해결 단서]
```

### 고클릭률 패턴 목록

| 패턴 | 예시 |
|------|------|
| [키워드] + "이거 모르면 손해" | "귀마개 추천 이거 모르면 손해예요" |
| [타겟 상황] + "보세요" | "층간소음 때문에 잠 못 드는 분들 보세요" |
| [문제] + "[숫자]가지 이유" | "멀미 패치 효과 없는 3가지 이유" |
| "왜" + [현상] | "왜 귀마개 써도 소음이 들릴까요?" |
| [키워드] + "후기" | "산행용 텀블러 3개월 사용 후기 (등산객 추천)" |
| [숫자] + "만에" + [결과] | "2주 만에 무릎 통증 잡은 방법" |
| [비교] + 의외의 결론 | "비싼 귀마개 vs 싼 귀마개 실험 결과" |
| "이거" + [상황] + "답" | "이거 하나로 생리통 끝난다는 게 사실이에요?" |

## 3. 금지 패턴 (하면 안 되는 네이밍)

### 회사 언어 사용 (가장 위험)
```
❌ 판매자가 강조하고 싶은 기능·제조 과정을 이름으로 씀

예) "치킨 철판이야" → 판매자는 철판 조리가 중요했지만
    고객은 "슬러시 맥주집"으로 기억
예) "특허기술 ○○귀마개" → 고객은 특허에 관심 없고 효과에 관심

해결책: 고객이 실제로 어떻게 부르는지 먼저 조사하라
```

### 입에 안 달라붙는 이름
```
❌ 영어+한국어 조합으로 발음이 어색
예) "Way Back Home" → 사람들은 "멈춘 시간 속"으로 기억

❌ 음절이 너무 많고 발음이 복잡
예) "에코프렌들리바이오헬스케어이어플러그"

해결책: 한국어 기준 3~5음절, 자연스럽게 발음되는 단어
```

### 신규 브랜드의 의미 없는 이름 (유형 1 남용)
```
❌ 스타트업이 구글·애플처럼 의미 없는 이름 사용
예) "랑데뷰 샴푸" (예쁘지만 기능 모름) → 하나로 샴푸에 패배

해결책: 신규 진입자는 유형 2 (포지셔닝 중심) 필수
```

### 광범위한 타겟을 향한 이름
```
❌ "모든 분들에게 좋은 ○○"
예) "좋은 귀마개", "프리미엄 텀블러", "최고의 찜질기"
→ 타겟이 없으면 아무도 "내 것"으로 느끼지 않음

해결책: 타겟 세분화 후 그 세계의 특수한 언어 사용
```

### 경쟁사가 이미 선점한 언어
```
❌ 1등이 쓰는 단어 그대로 따라가기
예) 이미 "하나로"라는 제품이 있는데 "하나로2" 같은 이름

해결책: 경쟁사가 선점하지 않은 빈 언어 찾아서 선점
```

## 4. 네이밍 설명 시 엠군 어투

### 추천 표현 (확언형)
```
"이게 답입니다"
"고객들은 이미 이렇게 부르고 있어요"
"이름 자체가 사용 설명서예요"
"이쪽으로 해야 됩니다. 이쪽이 훨씬 효율적이에요"
"네이밍만 바꿔도 다른 시장이 보여요"
"고객 언어 써야죠. 회사 언어 말고요"
"기억에 남는 별명이 돼야 됩니다"
```

### 실패 이름 지적 시
```
"이거는 자기들만 아는 이름이에요"
"고객한테 안 와닿죠"
"아무 의미 없잖아요, 고객 입장에선"
"이름이 싸운 거예요, 두 개가. 누가 이겼겠어요?"
"회사가 원하는 이름이랑 고객이 기억하는 이름이 달라요"
```

### 네이밍 원칙 강조 시
```
"여러분들, 유형 2로 가세요. 우리가 애플 같은 자신감이 있으면 좋지만."
"타겟 먼저, 이름 나중. 타겟 없이 이름 짓지 마세요"
"고객이 어떻게 기억하느냐가 전부예요"
"집단지성이에요. 혼자 짓지 말고 여러 명한테 물어보세요"
```

## 5. 관여도 맞춤 네이밍 표현

### 고관여 타겟 (많이 알고 많이 고민하는)
```
목적: 고민을 줄여주는 이름
접근: 이름에서 "더 이상 고민 안 해도 된다"는 신호
예) 하나로 샴푸 → "이제 두 개 살 필요 없어요"가 이름에서 전달됨

표현:
"[타겟이 이미 아는 전문 용어]를 이름에 넣어 신뢰감 형성"
"더 이상 고민 안 하셔도 됩니다. [이름]이 이걸 해결했거든요"
```

### 저관여 타겟 (별로 신경 안 쓰는)
```
목적: 관여도를 높이는 이름
접근: 이름이 궁금증을 만들거나 결핍을 자극
예) 돼지코팩 → 귀엽고 신기해서 클릭·구매 욕구 생김

표현:
"궁금하게 하는 이름 → 관여도를 끌어올린다"
"비주얼+이름이 동시에 욕구를 자극해야 함"
```
