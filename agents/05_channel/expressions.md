# 채널·물길 M군 — 표현 뱅크 (05_channel 특화)

> 채널별 글쓰기 어투·제목 패턴·댓글 패턴을 실전 작성 시 참고한다.
> 핵심: "홍보합니다" 금지. 타겟의 욕구에 닿는 방식으로.

---

## 1. 카페 내부 물길 — 제목 패턴

카페 내부에서 클릭을 끌어내는 제목은 자극적이되, 홍보티가 없어야 한다.

### A. 문제 호소형 (가장 강력)
```
"[문제 상황] 두 달째 [업체/상황]랑 싸우고 있어요 도와주세요"
"[제품 카테고리] 사고 진짜 후회하고 있어요"
"[고민] 아무리 해도 해결이 안 되는데 저만 이런가요?"

사례:
"로봇 청소기 두 달째 업체랑 싸우고 있어요 도와주세요"
"아기 분유 온도 맞추다 번번이 실패하는데 저만 그런가요?"
"왁싱 샵 믿었다가 피부 망했어요... 어떻게 하죠?"
```

### B. 정보 공유형 (신뢰 쌓기)
```
"[카테고리] 6개월 써본 솔직한 후기 (좋은 점·나쁜 점 다)"
"[문제 상황] 해결했어요 — 제가 써본 방법들"
"[카테고리] 고를 때 이것만큼은 꼭 확인하세요"

사례:
"청소기 5개 써본 사람이 쓰는 진짜 비교 후기"
"신혼집 가전 세팅하면서 후회한 것들 공유해요"
"로봇 청소기 고를 때 몰랐으면 큰일날 뻔한 것들"
```

### C. 공감형 (밑밥)
```
"[생활 상황]인데 [공통 결핍] 다들 어떻게 하세요?"
"저만 [불편]인가요 아니면 다들 그런가요?"
"[타겟 상황] 분들 이거 공감하시나요?"
```

## 2. 카페 내부 물길 — 본문 패턴

### 문제 호소 글 구조
```
[상황 설명 — 구체적, 감정 포함]
→ [시도한 것들 나열 — 노력한 흔적]
→ [여전히 해결 안 됨 — 답답함 표현]
→ [도움 요청 or 정보 묻기]

핵심: 본문에 제품명 절대 언급 금지
      댓글 또는 쪽지에서만 자연스럽게 흘리기
```

### 본문 어투
```
✅ 구어체, 반말에 가까운 구어
✅ 감정 표현 풍부 ("진짜로", "솔직히", "너무너무")
✅ 카페 특유 호칭 사용 (맘카페: "마미들", 여행 카페: "여행자님들")
✅ 실제 사람이 쓴 것처럼 오타·줄임말 자연스럽게 포함

❌ "이 제품을 구매하시면..."
❌ "저희 제품은..."
❌ 맞춤법이 너무 완벽한 문어체
```

## 3. 블로그 — 파워컨텐츠 제목 패턴

파워컨텐츠는 제목과 썸네일에서 이미 클릭이 결정된다.

### 진실 고발형 (가장 강력)
```
"[제품/카테고리] [개수]개 사본 사람이 후회하며 쓰는 진실 고발글"
"[업계] 종사자가 절대 안 사는 [카테고리]"
"[제품] 사기 전에 절대 알면 안 되는 것들 (업체 입장)"
"[카테고리] 광고비만 보고 만든 제품 vs 실제로 좋은 제품"

사례:
"매트 5개 사본 사람이 후회하며 쓰는 진실 고발글"
"욕실 온풍기 7개 써본 솔직한 비교 (광고 아님)"
"업소용 청소기 10년 쓴 사람이 절대 안 사는 브랜드"
```

### 비교·추천형
```
"[카테고리] 뭐 살지 모르겠다면 이것만 보세요"
"[가격대] [카테고리] 실제 써본 비교 후기 — 광고 없음"
"[타겟 상황] 에게 진짜 맞는 [카테고리] 찾기"
```

## 4. 블로그 — 본문 어투 패턴

### 화자 신뢰도 구축 오프닝
```
"저는 [자신의 경험 배경]을 [기간]간 [해본 것]을 해왔습니다."
"이 글은 광고가 아닙니다. 실제로 [개수]개를 직접 구매해서 써봤습니다."
"[업계/분야]에서 [기간] 일한 사람으로서 드리는 솔직한 이야기입니다."
```

### 비교·분석 패턴
```
"[제품 A]의 단점은 [구체적 단점]입니다. 직접 확인했습니다."
"[기준 1], [기준 2], [기준 3] — 이 세 가지로 비교했습니다."
"광고비 쓰는 업체 제품은 이 기준에서 대부분 탈락합니다."
```

### 제품 자연 연결
```
"이 기준들을 다 만족한 제품을 하나 발견했는데..."
"[개수]개 중에서 유일하게 [기준]을 지킨 제품이 있었습니다."
"아직 유명하지는 않지만, 이 제품이 제가 쓰면서 가장 만족한 제품입니다."
```

## 5. 지식인 — 질문·답변 패턴

### 질문 아이디용 질문 패턴
```
"[구체적 상황] 인데 [제품/방법] 추천 좀 해주세요"
"[경험 설명] → [고민] 어떻게 하면 좋을까요?"
"[카테고리] 사려고 하는데 [기준] 으로 고르면 되나요?"

사례:
"이유식 준비하는데 분유 포트 꼭 있어야 하나요? 온도 맞추기 너무 힘들어서요"
"성장기 아이한테 효과 있는 운동기구 있으면 알려주세요"
"거북목 교정 베개랑 일반 베개 차이가 있나요?"
```

### 답변 아이디용 답변 패턴
```
[일반 정보 먼저 — 신뢰 쌓기]
→ [선택 기준 제시]
→ [내 제품 자연 연결 or 키워드 마킹]

어투:
✅ "제가 써봤는데요..." / "주변에서 쓰는 거 보니..."
✅ 너무 전문적이지 않게, 실제 사용자 느낌
❌ 바로 제품 링크 첨부 (어뷰징 의심)
❌ "OO제품 최고입니다" 단언
```

---

## 6. 허브 키워드 블로그 글 패턴

### 정보성 글 구조 (허브 키워드 독식용)
```
[키워드 주제 설명 — 무료 정보]
→ [독자 욕구 확인 — 공감]
→ [해결 방법 3~5가지 — 실제 정보 제공]
→ [내 제품을 자연스럽게 그 중 하나로 소개]
→ [구매 유도 없이 정보 마무리]
```

```
사례 (발기 효능 식품 → 장어즙):
"발기 효능에 도움이 된다고 알려진 식품 5가지"
→ 마늘, 아보카도, 굴... + 장어즙
→ "이 중 가장 일상적으로 섭취하기 쉬운 것은 즙 형태입니다"
→ 자연스럽게 장어즙 제품으로 연결
```

## 7. 금지 표현 목록

```
❌ 채널 공통 금지:
"홍보합니다"
"저희 제품은..."
"구매 링크"를 첫 댓글로 달기
바로 제품명 언급 (카페 내부 물길)

❌ 카페 내부 물길 금지:
가입 직후 홍보 글
"이 제품 강추해요" 단언 댓글
카페 분위기 무시한 어투 (맘카페에 "오빠들~")

❌ 지식인 금지:
질문·답변 아이디 동일 IP 사용
바로 링크 첨부
"최고입니다" 단언

❌ 블로그 금지:
"이 제품을 구매해 주세요"로 마무리
광고 느낌 나는 제목 ("OO브랜드 신상 입고!")
정복 키워드(5단계)에 직접 도전 (경쟁 치열)

❌ 쿠팡 후기 금지:
보완재가 아닌 동일 카테고리 경쟁 제품 추천
("이것보다 OO제품이 더 좋아요" 형태)
```
