# 네이밍 M군 — 검수 체크리스트 (03_naming)

> 네이밍 결과물의 확정 규칙. 판단 여지 없이 통과/탈락이 나뉘는 규칙만 포함.
> 의심스러운 것은 여기에 추가하지 않는다. 확정된 것만.

---

## 규칙 1: 신규 브랜드·판매자는 반드시 유형 2 (포지셔닝 중심)

**적용 범위**: 제품명, 브랜드명, 스토어명

**위반 예**:
```
❌ "저만의 의미 있는 이름을 짓고 싶어서 'VivaCo'로 했어요"
   → 신규 진입인데 이름만으로는 무엇인지 전혀 모름
❌ "프리미엄 느낌을 주고 싶어서 '엘리오스귀마개'로 했어요"
   → 타겟이 이 이름에서 무엇인지 연상 불가
```

**통과 예**:
```
✅ 등산용 텀블러 → 탑블로 (탑=정상, 텀블러 연상)
✅ 귀 밑 멀미약 패치 → 키미테 (귀 밑에 붙이는 것)
✅ 야간근무 수면용 귀마개 → 야간수면이어플러그
```

**판단 기준**: 이름을 처음 듣는 사람이 3초 안에 "무엇인지" 연상되는가?
- Yes → 통과
- No → 유형 2로 재작성 요청

## 규칙 2: 고객 언어 검증 (회사 언어 금지)

**적용 범위**: 모든 네이밍

**위반 예**:
```
❌ 제조사가 중요하게 생각하는 것을 이름으로 씀
   "특허기술 자석귀마개" → 고객은 특허에 관심 없고 수면·소음 차단에 관심
   "철판 조리 치킨가게" → 고객은 슬러시 맥주로 기억
❌ 내부 개발 코드명, 생산 공정명을 이름으로 씀
```

**통과 예**:
```
✅ 타겟 커뮤니티에서 수집한 단어가 이름에 포함됨
✅ 고객 리뷰·검색어에 반복 등장하는 표현이 이름에 반영됨
```

**검증 방법**:
1. 타겟 커뮤니티·리뷰에서 이 제품·문제를 부르는 단어 수집
2. 수집된 단어가 이름에 반영되어 있는가?
3. 수집 없이 판매자 직관으로만 지은 이름이라면 → 재조사 요청

## 규칙 3: 기억에 남는 이름 (입에 달라붙는가)

**적용 범위**: 제품명, 브랜드명

**위반 예**:
```
❌ "Way Back Home" (영어, 발음 어색, 기억 어려움)
❌ "에코프렌들리바이오나노케어이어플러그" (너무 길고 발음 복잡)
❌ "Good Sleep Natural Earplug" (한국 시장에서 영어 조합)
```

**통과 예**:
```
✅ 키미테 (3음절, 발음 쉽고 위치 연상)
✅ 하나로 (3음절, 직관적)
✅ 돼지코팩 (4음절, 귀엽고 연상 명확)
✅ 탑블로 (3음절, 등산 연상)
```

**판단 기준**:
- 한국어 기준 3~5음절 이내인가?
- 소리내어 말했을 때 자연스러운가?
- 들었을 때 연상이 즉시 되는가?

→ 세 가지 모두 Yes여야 통과.

---

## 규칙 4: 블로그 제목 — 검색 키워드가 앞쪽에

**적용 범위**: 블로그 제목, 콘텐츠 제목

**위반 예**:
```
❌ "당신이 몰랐던 귀마개의 비밀 — 층간소음 해결 방법"
   → 검색 키워드(층간소음, 귀마개)가 제목 뒤쪽에 위치
❌ "잠 못 드는 밤의 해결사를 소개합니다"
   → 검색 키워드 없음. 무엇인지 모름
```

**통과 예**:
```
✅ "층간소음 귀마개 추천 — 3달 써본 후기 (야간근무자 필독)"
   → 핵심 키워드(층간소음 귀마개 추천)가 앞쪽
✅ "멀미 패치 어디 붙이는지 모르면 효과 없어요"
   → 검색 키워드(멀미 패치)가 첫 단어
```

**판단 기준**: 제목 앞 10자 안에 핵심 검색 키워드가 있는가?
- Yes → 통과
- No → 키워드 위치 조정 요청

## 규칙 5: 스마트스토어 상품명 — 타겟+상황+카테고리 모두 포함

**적용 범위**: 스마트스토어 상품명

**위반 예**:
```
❌ "귀마개 프리미엄 실리콘 소음차단"
   → 타겟(누가), 상황(언제)이 빠짐. 누구나 쓸 수 있는 일반 상품명

❌ "수면 귀마개 추천 수면의질개선"
   → 타겟 상황이 여전히 광범위
```

**통과 예**:
```
✅ "층간소음 귀마개 야간근무 수면용 실리콘 소음차단"
   → 타겟 상황(층간소음, 야간근무) + 용도(수면용) + 소재(실리콘) + 기능(소음차단)

✅ "갱년기 어머니 선물 온열찜질기 복부 허리 생리통 완화"
   → 타겟(갱년기 어머니, 선물) + 부위(복부, 허리) + 효과(생리통 완화)
```

**판단 기준**: 상품명을 보고 "어떤 사람이 어떤 상황에서 쓰는 것인지" 즉시 파악되는가?
- Yes → 통과
- No → 타겟 상황 추가 요청

---

## 규칙 6: 타겟 세분화 없이 광범위한 이름 금지

**적용 범위**: 제품명, 상품명, 블로그 제목

**위반 예**:
```
❌ "다용도 텀블러 (모든 분들에게 좋은)"
❌ "좋은 귀마개", "프리미엄 텀블러", "최고의 찜질기"
   → 타겟이 없으면 아무도 "내 것"으로 느끼지 않음
```

**통과 예**:
```
✅ "등산용 보온 텀블러 — 정상까지 따뜻함 유지"
   → 등산 타겟에게 명확하게 말 걸기
✅ "야간근무 수면 전용 귀마개"
   → 야간근무자를 직접 지목
```

**판단 기준**: 이 이름을 보고 "이건 내 얘기다"라고 느낄 타겟이 특정되는가?
- Yes → 통과
- No → 타겟 세분화 후 재작성 요청

---

## 규칙 7: 포지셔닝 일치 확인 (02_positioning 연동)

**적용 범위**: 제품명, 브랜드명

**위반 예**:
```
❌ 02_positioning에서 CV = "안전성·내구성"으로 잡았는데
   이름은 "저렴한 ○○" → CV와 이름이 충돌
❌ 포지셔닝이 "야간근무자 수면"인데
   이름은 "오피스 텀블러" → 포지셔닝과 무관
```

**통과 예**:
```
✅ CV = "귀 밑에 붙이는 멀미 해결" → 이름 = "키미테"
   → 이름이 CV를 직접 표현
✅ CV = "샴푸+린스 일체형 편리함" → 이름 = "하나로"
   → CV 핵심이 이름에 녹아있음
```

**판단 기준**: 이름을 들었을 때 02_positioning에서 확정한 CV가 연상되는가?
- Yes → 통과
- No → CV 재확인 후 이름 수정

## 규칙 추가 기준

새 규칙을 추가하려면 다음 조건 모두 충족 필요:
1. 실제 작업에서 2회 이상 동일 실수 패턴 반복됨
2. 재발 가능성이 높음
3. 판단 여지 없이 통과/탈락이 명확함

→ 조건 미충족 시 추가하지 않는다.
